##  주요 학습 사항

### . 병렬 에이전트 활용
개 에이전트를 동시에 실행하여 작업 시간을 크게 단축
- 보안 수정 개 + 기능 구현 개  **동시 진행**
- 예상 시간 시간 → 실제 시간 **분**

### .  통합 패턴
- 환경변수로  키 관리
- 재시도 메커니즘 구현
- 캐시로 비용 절감

### . 플러그인 아키텍처
-  인터페이스로 일관성 유지
- 동적 로딩으로 확장성 확보
- 메타데이터 기반 관리
