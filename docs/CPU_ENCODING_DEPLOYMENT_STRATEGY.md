### . 품질 허용 기준

**  기준**
- 최소 품질 % () → 시청 가능하나 권장 안 함
- 권장 품질 **%** () → 모바일 시청 충분 
- 최적 품질 % () → 고품질 요구 시

**크루즈닷  타겟 기준**
-  % 품질  **모바일  시청 충분**
