#  사용 가이드

## 개요

는 동시에 -개의 영상을 병렬로 생산하는 고성능 배치 처리 시스템입니다.

### 주요 기능

- **  기반 병렬 처리** . 사용
- **자동  수 조정**  코어 수 기반 -개 자동 계산
- **우선순위 큐 관리**      순서
- **에러 복구 로직** 최대 회 재시도 +   (초, 초, 초)
- **리소스 모니터링** 메모리/ 사용률 추적, % 초과 시 자동 스로틀링
- **  ** 회 실패 작업 자동 분리 및 저장
- **진행 상황 추적**  프로그레스바
- ** ** / 핸들링
