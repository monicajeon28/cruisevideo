## 수정 내역

### -- 자막 크기 및 줄바꿈 수정

#### . /. - () 기본값 변경
**위치**  

**변경 전**

 ( ,  ,    ) - 
    """
    - 기본값 → (.와 일치)
    ...
     한 줄당 글자 수 (기본 자,  폰트 기준)
    """
       *   # 자


**변경 후**

 ( ,  ,    ) - 
    """
    -- 기본값 → (강제 줄바꿈,   폰트 잘림 방지)
    ...
     한 줄당 글자 수 (기본 자,   폰트 기준)
    """
       *   # 자


**효과**
- 한 줄당 글자 수 자 → 자
- 줄 최대 자 → 자
- 강제 줄바꿈으로 가로 폭 초과 방지

### --  자막 타이밍 동기화

#### . . - ()  타이밍 수정
**위치**  -

**변경 전**

 ,   (.('', ))
     .('')  ''
        #  -  자막 복원 - 초 임팩트 문구 표시
          .('', .('', ''))
        .(
            '' ,
            '' .,
            '' ,           # .초 고정
            '' ,      # .초 고정
            '' ''
        )
               # .초 고정
        


**변경 후**

 ,   (.('', ))
     .('')  ''
        # --  자막 타이밍  사용 (.초 고정 해제)
              ()  
          .('', .('', ''))
        .(
            '' ,
            '' .,
            '' ,      #  실제  사용
            '' ,  #  실제  사용
            '' ''
        )
            #  실제  사용
        


**효과**
-  자막  .초 고정 →  사용 (예 .초)
-  오디오와 자막 타이밍 완벽 동기화
- 다른 세그먼트는 기존 로직 유지 (변경 없음)

### .  자막도 자/줄 적용
-  은 전역 설정

-  폰트  →  (-%)
- 시각적 임팩트 감소 가능성
- 대신 **가독성 향상** (잘림 없음  큰 글자)
