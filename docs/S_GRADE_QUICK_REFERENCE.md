## 실행 로드맵

###   (오늘, 시간) → .점 + 등급
 작업  시간  효과  누적 점수 
-----------------------------
 --   완료  +.점  .점 
 **--  **  **분**  **+.점**  **.점**  

###   (내일, 시간) → .점 등급
 작업  시간  효과  누적 점수 
-----------------------------
 **-- 전체**  **시간**  **+.점**  **.점**  
 **--**  **.시간**  **+.점**  **.점**  
 **--**  **시간**  **+.점**  **.점** (정규화) 

###   (이번 주, 시간) → 품질 강화
 작업  시간  효과

 --  .시간  이미지/비디오 교차 (다양성 +%)
