# - - 자동 검증 파이프라인 - 완료 요약

## 프로젝트 개요

영상 생산 즉시 - 여부를 자동으로 판정하고, 승인/거부/수동검토를 결정하는 완전 자동화 시스템을 구축했습니다.

## 주요 기능

### . - 자동 검증

**특징 벡터 (차원)**
-  강도, 길이, 타입 점수 (개)
-  평균 장면 길이, 분산, 부드러움 (개)
-  시작/피크/끝/분산 (개)
-  초/초/초/평균 (개)
-  전환/효과/컬러 (개)
-  //믹싱 (개)
-  총 길이, 장면 수, 텍스트, 컷 빈도, 다이나믹 레인지 (개)

**출력**


  "" "",
  "" .,
  "" .,
  "" 
    "" .,
    "" .,
    "" .,
    "" .,
    "" .,
    "" .
  



### . 자동 승인/거부 로직

 조건  -  신뢰도  결과  액션 
-----------------------------------
 최우수  ≥%  ≥%   자동 승인  즉시 배포 
 양호  -%  %   수동 검토  대시보드 확인 
 미달  %  -   재생산  큐 추가 

### . 재생산 큐

**우선순위 체계**

 - 긴급 (- %+, 수동 거부)
 - 일반 (- -%)
 - 낮음 (- % 미만)


**재시도 제한**
- 최대 회
-   (초 → 초 → 초)
- 회 실패 시    이동

### . 대시보드 

**엔드포인트**

 /&

# 수동 승인
 ///
 "" ""

# 통계 조회

# 재생산 큐 조회
