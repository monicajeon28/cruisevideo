##    (건) - 즉시 수정 필수

### - 허위 수치 잔존 및 자기모순 (건)

#### - .- -  프롬프트에 "%" 잔존
**문제**  프롬프트에 "%가 크루즈 처음입니다" 가 곳에 그대로 남아있음. - 목적(허위 수치 제거)와 정면 충돌.
**영향** 가 이 프롬프트를 받아 생성하면 영상에 "%" 문구가 그대로 들어감.
**수정** 해당 "패턴  의심 해소" 섹션에서 "%"를 제거하고 검증 가능한 표현으로 교체 (예 "처음이신 분이 대부분입니다")

#### - .- - () 내부에 "", "%" 하드코딩
**문제**   검증에 "", "%" 키워드가 포함되어 있어, 이 허위 수치가 없으면  점수가 점.
**영향** 등급 자체 검증이 허위 수치 존재를 요구함.
**수정**  키워드를 "년", "억", "시간", "케어", "보험", "경력" 등 검증 가능한 사실로 교체

#### - .- -  검증이 "명"을 여전히 요구
**문제** 에 '명' (.('명...', ))가 남아있고, 누락 시 "구체적 숫자 명 명시" 제안을 출력함. .에서 선착순 명을 금지패턴으로 추가했으므로 **자기모순** 한쪽은 금지, 한쪽은 요구
**수정** 에서 '명' 체크 제거. 긴급성은 한정표현과 자연적 희소성(시즌/일정 기반) 개 검증으로 변경

#### - . - 에 "선착순"이 여전히 긍정 검증 대상
**문제** .('선착순한정...') 에서 "선착순"이 포함됨. .에서 에서 "선착순"을 제거했고, 의  카테고리에도 "선착순"이 금지어. 한쪽은 금지, 한쪽은 통과
**수정** 의 한정표현 에서 선착순 제거

#### - .- - 에 "만 분 이상", "만 명" 패턴 잔존
**문제**  카테고리의 에 "만*분*이상", "만*명", "수만*분"이 여전히 있음. .에서 "+만*분*이상" 를 으로 선언했으므로 **자기모순** 금지어가  매칭 패턴으로 사용됨
**수정**   에서 "만 분/명" 관련 패턴 개 제거. "년", "년", "경력", "전문가" 만 유지

#### - . - 에 "만 분 이상" 허위 수치 포함
**문제**  부족 시 제안 메시지 "신뢰요소 추가 '년간 만 분 이상 모시면서, 시간 한국어로 케어해드려요'" 가 검증 불가 수치를 사용자에게 안내함
**수정** "년간 크루즈만 안내해왔고, 시간 한국어로 케어해드려요" 로 교체

##  핵심 문제 가지

### . 허위 수치 제거 미완료 (자기모순)
-  프롬프트에 "%" 잔존 (-)
-  가 "명" 요구  가 금지 (-, -)
- 에 "만 명" 매칭  에서 금지 (-)
- .에 "% 성공", ",명" 활성   충돌 (-)

**영향** 등급 무한 루프 (번 재시도 후 실패) + 허위 마케팅 콘텐츠 생산 위험

### . 상품-기항지 필터링 버그
- 가  를 한글명으로 변환 없이 검증 (-) → 모든 기항지 경고
-  매핑 중 정의 + 불일치 (-, -, -) →  
-  코드 누락 개 (-) → 이미지/ 매칭 실패

**영향** 상품 없는 기항지로 영상 생성 + 사용자에게 판매 불가능한 콘텐츠 제공

### . 테스트 건 즉시 실패
-  메시지  값 불일치 (-)
- 미존재 함수/파라미터  (-)
-  타이밍 기대값 불일치 (-)

**영향** / 파이프라인 즉시 실패 + 개발 흐름 차단
