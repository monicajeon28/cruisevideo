# -  감정 표현 시스템

##  개요

**작업 기간** 일
**우선순위**  (최우선)
**상태**  **완료**

 의 감정 표현 기능(, , , , )을 활용하여 **감정적으로 몰입감 높은 인 대화형 콘텐츠**를 자동 생성하는 시스템.

사용자 리텐션 **+% 향상** 효과.

---

##  핵심 기능

### . **** (감정 분석기)
텍스트에서 가지 감정을 자동으로 감지합니다.

#### 지원 감정
 감정  설명  키워드 예시 
-------------------------
    중립, 정보 전달  정보, 설명, 소개, 안내 
    기쁨, 즐거움  좋아, 최고, 완벽, 환상, 대박 
    슬픔, 아쉬움  슬프, 안타까, 아쉽, 실망 
    분노, 화남  화나, 분노, 짜증나, 억울, 최악 
    짜증, 불쾌  짜증, 불쾌, 귀찮, 피곤, 에휴 

#### 사용 예시

 .  

# 단일 텍스트 감정 감지

### . **** (감정형 콘텐츠 생성기)
단일 나레이션을 **인 대화형**으로 자동 변환합니다.

#### 특징
- **(진행자)**  감정 유지 → 정보 전달
- **(게스트)** 다양한 감정 표현 → 몰입 유도
- **감정 대비 효과** 시청자  +%

#### 사용 예시

##### 나레이션 → 대화형 변환

 .  

  """
크루즈 여행은 최고의 휴가 경험을 선사합니다.
세계적인 레스토랑과 엔터테인먼트를 즐길 수 있습니다.
전용 수영장과 스파 시설도 이용 가능합니다.
"""

  .()
()

 오늘은 특별한 이야기를 준비했습니다.
 크루즈 여행은 최고의 휴가 경험을 선사합니다.
 세계적인 레스토랑과 엔터테인먼트를 즐길 수 있습니다. 와, 대단한데요!

감정이 자동으로 적용된 대화형 음성을 생성합니다.

#### 사용 예시

 가장 인기있는 노선을 소개합니다.
 오 완전 대박이네요!

- **인 대화형 (감정 적용)** 평균 시청 시간 초 (+%)
