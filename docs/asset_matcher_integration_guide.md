## . 실전 사용 예시

### 시나리오 "크루즈 객실 가이드" 영상 생성

#### 나레이션

" 벨리시마의 객실에는 넓은 발코니가 있어
아름다운 바다를 감상할 수 있습니다.
객실 내부는 현대적이고 편안한 인테리어로 꾸며져 있습니다."


#### 기존 시스템 결과

  골든파티  벨리시마. (랜덤)
   벨리시마 객실 내부. (객실 키워드)
   바다 파도. (바다 키워드)
   인테리어. (인테리어 키워드)

 문제 이 파티 영상 (객실과 무관)

#### 개선 시스템 결과

  벨리시마 객실 전경 외관. (객실 테마 매칭)
   벨리시마 발코니 오션뷰. (발코니 + 바다 근접)
   벨리시마 객실 내부 인테리어. (내부 + 인테리어 근접)
   벨리시마 발코니 석양. (발코니 서브 카테고리)

 개선 모든 영상이 "객실" 테마에 일치

## . 주의사항

### .  매핑 파일 업데이트 필요
현재 .의 키워드가 너무 단순합니다.

**개선 필요**


  "" " 벨리시마 객실 내부.",
  "" "",
  "" "객실", "내부", ""  // 현재


→


  "" " 벨리시마 객실 내부.",
  "" "",
  "" "",  // 추가
  "" "객실", "내부", "침대", "", "인테리어", "", "벨리시마",
  "" "발코니", "외부"  // 추가



### .  폴더 정리
현재  폴더에 공연/파티 영상만 있으면 객실 테마에 맞는  선택 불가

**권장 구조**

/
  /
    -  벨리시마 객실 전경.
    - 로얄 스위트 외관.
  /
    -  골든파티.
    - 뮤지컬 공연.
  /
    - 뷔페 전경.
    - 정찬 레스토랑.

## . 

### . 기존 를 완전히 교체해야 하나요
**.** 아니요. 는 보완 도구입니다. 기존 시스템을 사용하되, 추가로 문맥 필터링만 적용합니다.

### . 성능 저하가 걱정됩니다.
**.** 문맥 분석은  정도 추가되지만, 전체 영상 생성 시간(수십 초)에 비하면 미미합니다.

### .  파일을 모두 수정해야 하나요
**.** 선택적입니다. 우선 자주 사용되는 상위 개 영상만 수정해도 효과가 큽니다.
