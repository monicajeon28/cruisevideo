## 위험 완화 전략

### 리스크   % 이하 (보수적 미달)

**증상**
- 프로필 클릭 % 이하  월 예약 건, 월 수익 만원
- 손익분기 개월 이상

**원인 가설**
.  초 도파민 강도 부족 (가격 충격 미흡)
.  타겟팅 오류 (알고리즘이  노출)
. 썸네일  저조 (텍스트 가독성 부족)

**대응안**
- **즉시 (주)**  전용 폴더 우선 선택 강화 (현재 개 → 개 확장)
- **단기 (주)** 가격 숫자   크기 배 확대 (.초 → .초 유지)
- **중기 (개월)** 대표자 실제 얼굴 영상 주 편 병행 ( 신뢰도 보완)

### 리스크  랜딩  % 이하 (플랫폼 마찰)

**증상**
-  진입 %  프로필 클릭 ,명 → 실제 랜딩 ,명

**원인 가설**
. 링크 클릭 단계 마찰 ( 앱 → 브라우저 → )
. 전화번호 워터마크 가독성 부족
.  세대  채널 사용법 미숙

**대응안**
- **즉시** 전화번호 워터마크 폰트 크기 .배 확대 + 하이라이트 색상 변경
- **단기** 영상 설명란 "링크 클릭 방법" 스크린샷 장 추가
- **중기** 롱폼 영상에서 "카카오톡 채널 검색 방법" 시연 (화면 녹화)

### 리스크  상담→예약 전환 % 이하 (  특성)

**증상**
- 예약 전환 %  월 예약 건, 월 수익 만원

**원인 가설**
. 영상 신뢰도 부족 ( + 얼굴 없음)
. 가격 불안 (영상 가격  실제 가격 괴리)
. 경쟁사 비교 후 이탈

**대응안**
- **즉시** 상담 스크립트 강화 ("영상 보셨나요" → 영상 맞춤 제안)
- **단기** 실제 고객 후기 수집 시스템 구축 (영상 → 후기 영상 순환 구조)
- **중기** 기존 고객  활용 (년 만팀 → 영상 공유 부스트)
