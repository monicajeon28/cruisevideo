## 산출물 목록

### . 메인 리포트 (개 문서)

####  . (, 줄)

**내용** 가지 문제에 대한   분석 + 근본 원인 + 예방 방법

**구조**
-  
- 문제  개별 분석 ( )
- 근본 원인 종합 (가지 패턴)
- 예방 가능성 분석
- 즉시 실행 권장사항
-   연기 항목

**핵심 발견**
- 패턴  작업 스코프 불일치 ( 인계 누락)
- 패턴  수동 변수 정의 방식 (자동화 부재)
- 패턴  복잡도 예측 실패 ( → )

---

####  . (, 줄)

**내용** 즉시 실행 체크리스트 (시간 작업)

**구조**
- 검증 완료 항목 (자동 변수 추출 스크립트)
- 즉시 실행 필요 항목 (. 수정 +  개)
-   연기 항목
- 완료 체크리스트

**핵심 작업**
. .에 ,  변수 추가 (분)
.  파일 개 생성 (, , ) (시간)
. 검증 스크립트 재실행 (분)

---

####  . (, 줄)

**내용** 경영진 요약 ( )

**구조**
- 핵심 발견 사항 (페이지)
- 근본 원인 가지 패턴
- 예방 가능성 평가 ( 분석)
- 즉시 실행 권장사항
- 핵심 교훈 ( )

**핵심 지표**
- 평균 예방 확률 %
- 투자 시간 .시간
-  시간당 .개 문제 방지

**실행 방법**

**출력 예시**

  '', '', '', ...

 문제 발견 템플릿에는 있지만 .에 없는 변수
   -  (개 파일에서 사용)
   -  (개 파일에서 사용)

 미사용 규칙 개
   - , , , ...


**활용**

- / 파이프라인 자동 검증용
-  간 인계 시 변수 일치 검증
