  -  **영상 편 조회수**  **,회+**    /  카테고리  우선

 ** **  **상담 신청 (누적 주)**  **건+**    ****  **대표자 얼굴 영상 % 상향** 

**  판정 기준**

**  판정 기준**

-  ** (%)** 실제 고객 후기 수집 시스템 구축 (영상 → 후기 영상 순환)
-  ** (% 이하)** 기존 고객  활용 (년 만팀 → 영상 공유 부스트)

## 위험 완화 전략 (리스크별)

### 리스크   % 이하 (보수적 미달)

#### 증상


프로필 클릭 % 이하
→ 월 ,명 클릭 (,회  %)
→ 월 예약 건
→ 월 수익 만원
→ 손익분기 개월 이상


#### 원인 가설 (우선순위)

 순위  원인  확률  검증 방법 
---------------------------
    초 도파민 강도 부족  %   초 유지율 측정 (목표 %) 
    타겟팅 오류 (알고리즘이  노출)  %    시청자 연령대 분석 
   썸네일  저조 (텍스트 가독성 부족)  %  썸네일 / 테스트 (가격 숫자 크기 비교) 

#### 대응안 (단계)

 단계  시간  작업  기대 효과 
---------------------------
 **즉시 (주)**     전용 폴더 우선 선택 강화 (개 → 개 확장)   +% 
 **단기 (주)**    가격 숫자   크기 배 확대 (.초 → .초 유지)   +.% 
 **중기 (개월)**    대표자 실제 얼굴 영상 주 편 병행 ( 신뢰도 보완)   +% (신뢰도 상승) 

#### 실행 체크리스트

-    전용 폴더 개 확장 (동유럽/알래스카 기항지 중심)
-   .  . → .초 수정
-   대표자 촬영 스크립트 작성 (초 인트로 + 크루즈 장점 가지)
-   썸네일 / 테스트 종 (가격 숫자 크기 대형  중형)
-     시청자 연령대 분석 (목표 대 이상 %+)

### 리스크  랜딩  % 이하 (플랫폼 마찰)

#### 증상


 진입 % 이하
→ 프로필 클릭 ,명 → 실제 랜딩 ,명
→ 월 예약 건
→ 월 수익 만원


#### 원인 가설 (우선순위)

 순위  원인  확률  검증 방법 
---------------------------
   링크 클릭 단계 마찰 ( 앱 → 브라우저 → )  %  링크 클릭률 측정 (목표 %) 
   전화번호 워터마크 가독성 부족  %  상담 시 "어디서 번호를 보셨나요" 설문 
    세대  채널 사용법 미숙  %  상담 시 " 채널 어렵지 않으셨나요" 설문 

#### 대응안 (단계)

 단계  시간  작업  기대 효과 
---------------------------
 **즉시**    전화번호 워터마크 폰트 크기 .배 확대 + 하이라이트 색상 변경 (흰색→노란색)   +% 
 **단기**    영상 설명란 "링크 클릭 방법" 스크린샷 장 추가   +% 
 **중기**    롱폼 영상에서 "카카오톡 채널 검색 방법" 시연 (화면 녹화 초)   +% 

#### 실행 체크리스트

-   .   →  수정
-   워터마크 하이라이트 색상  →  ( , , )
-   영상 설명란 템플릿 추가 (스크린샷 장 링크 클릭 →  열기 → 채널 진입)
-   롱폼 영상   "상담 방법" 화면 녹화 시연 (초)
-   상담 시 설문 항목 추가 ("어디서 번호를 보셨나요", " 채널 어렵지 않으셨나요")

### 리스크  상담→예약 전환 % 이하 (  특성)

#### 증상


예약 전환 % 이하
→ 상담 명 → 예약 건
→ 월 수익 만원


#### 원인 가설 (우선순위)

 순위  원인  확률  검증 방법 
---------------------------
   영상 신뢰도 부족 ( + 얼굴 없음)  %  상담 시 "영상 신뢰도" 설문 (점 척도) 
   가격 불안 (영상 가격  실제 가격 괴리)  %  상담 시 "가격 예상 범위" 설문 
   경쟁사 비교 후 이탈  %  상담 시 "다른 여행사 상담 여부" 설문 

#### 대응안 (단계)

 단계  시간  작업  기대 효과 
---------------------------
 **즉시**    상담 스크립트 강화 ("영상 보셨나요" → 영상 맞춤 제안)  전환율 +% 
 **단기**    실제 고객 후기 수집 시스템 구축 (영상 → 후기 영상 순환 구조)  전환율 +% 
 **중기**    기존 고객  활용 (년 만팀 → 영상 공유 부스트)  전환율 +% (신뢰도 상승) 

#### 실행 체크리스트

-   상담 스크립트 템플릿 작성 (영상 종별 맞춤 제안)
-   실제 고객 후기 수집 양식 (이름+나이+크루즈 일정+만족도 점)
-   후기 영상 촬영 가이드 (초 인터뷰 형식,  부부 중심)
-   기존 고객  이메일 발송 (영상 공유 부탁 + 추천 할인 만원)
-   상담 시 설문 항목 추가 ("영상 신뢰도", "가격 예상 범위", "경쟁사 상담 여부")

. 즉시 중단 새 영상 업로드 중단 (기존 영상 유지)
. 원인 분석 일 이내 원인 가설 가지 도출
. 대응안 실행 주 이내 대응안 단계 실행 (즉시+단기+중기)
. 재측정 주 후 재측정 (목표  이상 회복)
. 재개  달성 시 영상 업로드 재개
