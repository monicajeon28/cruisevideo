## 예상 결과

### 완주율 변화
 시점  완주율  증가율  누적 증가 
---------------------------------
 현재  %  -  - 
   완료  .%  +.%  +.% 
   완료  .%  +.%  +.% 
   완료  **.%**  +.%  **+.%** 

###  계산
- **투입 시간** 시간
- **완주율 증가** +% (% → %)
- ** 증가 예상** +% (.% → .%)
- **월 수익 증가 예상** 만원 → 만원 (+만원/월)
- **연 수익 증가** +,만원
- **** **,만원 / 시간**  **만원/**

### 장기 효과 (주 후,  - 완료 시)
- 완주율 % → **%** (+%)
- 월 수익 만원 → **만원** (+%)
- 등급 % 근접
