# 스토리텔링 구조 분석 알래스카 라이브 방송 (분)

**분석 대상** 알래스카 라이브 방송 스크립트 .
**총 시간** 분 초 (약 분)
**발화자** 참석자  (모니카), 참석자  (맥스)
**포맷** 인 대화형 라이브 방송 (/네이버 라이브)

## . 시간대별 구조 (- )

### -분  + 신뢰 구축 ( )
**핵심 전략** 빠른  → 즉시 가치 제시

 시간  내용  기능 
------------------
 -  인사 + 지난주 상기 + 오늘 주제 예고 (로열 보이저호 + 알래스카)  - 
 -  **빙하 긴급성 ** "빙하가 녹는다", "언젠간 꼭 내가 알래스카 간다"   +  
 -  **대 주목 포인트 제시** () 만 년 빙하, () 패키지 대비 만원 절약, () 자유 여행 일정    
 -  로열 캐리비안  "세계 등급 크루즈" 포지셔닝    

**패턴**
- **문제 제기 → 해결책 제시 → 권위 설정** (분 안에 완료)
- 감정 자극 ("빙하가 녹는다") → 논리 ("만원 절약") → 신뢰 ("로열  짱")

### -분 정보 밀도 최고점 (  )
**핵심 전략** 여행 루트 + 시각 자료 + 인솔자 꿀팁

 시간  내용  밀도 
------------------
 -  **루트 설명** 시애틀 → 주노 → 스케그웨이 → 앤디콧암 → 빅토리아 → 시애틀   (지도 활용) 
 -  **경쟁사 비교** 다른 알래스카 크루즈  로열 보이저호 (아이 프로그램, 날씨 준비, 기항지 투어 부족)   (차별화) 
 -  **보이저호 선택 이유** 남녀노소 즐김, 서핑/골프/스케이트,  리모델링   (시각 자료) 
 -  **음식 소개** 정찬  뷔페, "원모어" 팁, 빙하 감상하며 식사   (실용 정보) 

**패턴**
- **지도 → 사진 → 영상** 순서로 시각 자료 강화
- **"인솔자 년 경력이 알려준 꿀팁"** 반복 강조 (신뢰 앵커)
- 분대  **정보 과부하 구간** (의도적 → 나중에 재후킹 필요)

### -분 감정 자극 + 체험 시뮬레이션 ( )
**핵심 전략** "내가 거기 있다면" 시나리오 + 영상 자극

 시간  내용  감정 
------------------
 -  **보이저호 내부 즐길거리** 정찬, 뷔페, 자쿠지, 브로드웨이 공연, 스케이트  흥분 + 기대 
 -  **영상 편 재생** (보이저호 홍보 영상 분)  감각 자극 
 -  **주노 기항지** 빙하 트레킹, 고래 관찰, 카약, 헬기 투어  모험 +     
 -  **시애틀 가는 법** 공항 → 경전철 → 호텔 (사진 첨부)  실용성 + 안전감 
 -  **시애틀 관광** 파이크플레이스 시장, 스페이스니들, 스타벅스 호점  보너스 가치 

**패턴**
- **영상 활용**  대화 휴식 + 감각 자극 (분 영상  단어 대체)
- **"나 갈 거예요" 반복** (참석자 이 회 이상) → 청중 동일시
- **"인솔자가 알려준 꿀팁"** 계속 강조 (, , )

### -분 재후킹 +  강화 (- )
**핵심 전략** 실패 사례 → 해결책 → 가격 공개 직전 긴장감

 시간  내용  기능 
------------------
 -  **크루즈 승선 가이드** 번 부두 주의, 우버 택시 가격, 안전 교육 필수  실용 정보 () 
 -  **영상 편 재생** (빙하 + 주노 관광 분)  감각 재자극 
 -  **주노 액티비티** 고래 투어, 카약, 헬기+개썰매 (가격 언급 시작)  추가 가치 제시 
 -  **음식 추천** 킹크랩, 연어 구이, 순록 소시지  감각 자극 (식욕) 
 -  **영상 편 재생** (스케그웨이 레일웨이 분)  감정 피크 (눈물 언급) 
 -  **빅토리아 관광** 주의사당, 부차트 가든, 마차 투어  보너스 가치 

**패턴**
- **-분  가장 긴 구간** (분) → 가격 공개 직전 가치 극대화
- **영상 편 (총 분)**  시청자 피로 완화 + 감정 리셋
- **"눈물 나더라", "단 한 명도 안 좋다고 한 사람 없다"** (사회적 증거)

-             "빙하가 녹고 있어요. 언제 또 가보겠어요" + 빙하 비주얼
-            "패키지 만원 → 우리 만원 (만원 절약)"
-           빙하 근접 + 고래 + 스케이트장 (컷 빠른 전환)
-            "로열 캐리비안 등급 크루즈 + 년 인솔자 꿀팁"
-            "인 만원 (월 만원), 만원 지원 (선상팁+항만세)"
