## -- 가격 범위 검증 추가 (분)

### 문제 분석

**파일** //.
**** 

**현재 코드**

 ()
    """가 "만원" 형식으로 치환되는지 확인"""
    
          .("", "", "")
     
        .("템플릿 파일 없음")

      .(, )

    #  문제 비정상 값도 통과 ("만원")
     .('+만원', ), 
        " 치환 실패 '만원' 패턴 없음"


**위험 시나리오**
-  "만원" →  (잘못됨)
-  "만원" →  (잘못됨)
-  "만원" →  (정상)

---

### 수정 코드


 ()
    """가 "만원" 형식으로 치환되는지 확인"""
    
          .("", "", "")
     
        .("템플릿 파일 없음")

    #  수정  만 검증
      
       .("", )
        .(.("", ""))
        .(.("", ""))

      " ".((, ))

    #  수정  정교한 패턴 + 범위 검증
      '(,)만원'  # 만원 (자리)
      .(, )

     , (
        " 치환 실패 '만원' 패턴 없음"
        "출력 텍스트 "
    )

    #  수정  합리적인 범위 확인
      (.())
         , (
        "비정상 가격 범위 만원"
        "허용 범위 만원"
        "에서 값 확인 필요"
    )

    #  수정  티어별 가격 범위 검증 (선택)
      .("", ).("")

       ""
             , 
            " 가격 범위 초과 만원 (기대 만원)"
       ""
             , 
            " 가격 범위 초과 만원 (기대 만원)"
       ""
             , 
            " 가격 범위 초과 만원 (기대 만원)"


**변경점**
. + → , (자리 숫자만)
. 가격 범위 검증 추가 (만원)
. 별 세부 범위 검증 (선택)
. 에러 메시지 상세화
