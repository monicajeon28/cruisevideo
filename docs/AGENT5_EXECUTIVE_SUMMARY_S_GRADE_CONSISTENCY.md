## 대  

### .   필드 누락 (% 실패율)

 누락 개 (-.점/개)
- 누락 개 (-.점/개)
금지어 생성 개 (-.점/개)
  누락 개 (-.점/개)

→ 실제 실패 개 영상 (%)


**원인** .는 템플릿 기반, 실제  호출은 .에서 발생 (추정)

**해결책**   사후 수정 시스템
- () 함수로 필드 강제 보정
- 실패율 % → **%**

### . 대 후반 긴급성 거부 (% 전환 하락)

**문제**  긴급성 단계 ("명 남았습니다")
- 대 초반 효과 ↑ ( 민감)
- **대 후반** 거부 ↑ ("압박감", "사기 같다" %)

**심리학적 원인**
-   (손실 회피)   (놓침 공포)
- 보수주의 성향 강함

**해결책** 타겟 세분화 제거 → 범용 프레임

기존 (대 후반 전용)
"년 경력, 억 보험 포함, 편하게 문의하세요"
→ 긴급성 없음, 클릭률 -%

개선 ( 범용)
"년 경력 만 명 선택, 카카오톡 '크루즈닷' 검색.
명 한정 혜택 있습니다" (긴급성 마지막 배치)
→ 대 긴급성 캐치, 대 신뢰 캐치


**효과** 대 후반 달성률 % → **%**
