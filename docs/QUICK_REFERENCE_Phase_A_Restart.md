##   진행 순서 ( )


분   - 
  ├─ . 실행
  ├─ 누락 파일 생성 (있으면)
  ├─  
  └─ . 업데이트

시간   . .
  ├─  개 조합
  ├─  개 변수
  ├─ . 실행 (검증)
  ├─  체크리스트 확인 (%)
  ├─  
  └─   검증 (점 이상)

시간   .
  ├─  단위 테스트 개 작성 먼저 ()
  ├─  클래스 구현 ()
  ├─ 리팩토링 ()
  ├─  체크리스트 확인 (%)
  ├─  
  ├─ . 업데이트
  └─   검증 (/ 통과)

시간   대본 엔진 연동

  ├─ 통합 테스트 개 영상 생성
  ├─  체크리스트 확인 (%)
  ├─  
  └─   검증

시간   통합 테스트
  ├─ 개 조합 영상 생성
