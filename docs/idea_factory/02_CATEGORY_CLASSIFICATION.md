#   카테고리 분류 프레임워크 ( 스코어 기반)

**작성일** --
**역할** 상위 .% 마케팅 전략가 (카테고리 분류 전문가)
**목표** 개 카테고리를 소구점 깊이별로 재분류하고 하루 개 생산 시 최적 배분 전략 수립

## .  

### 핵심 발견 가지
. **정보 깊이형 ( ) 카테고리가 등급 달성률 .배 높음**
   - 기항지정보/선내시설/꿀팁 등급 점+ 달성률 %
   - 감정 세일즈형 등급 달성률 % (도파민 과잉 → 완주율 저하)

. **하루 개 생산 시 - 배분 최적 비율 **
   -   (정보 깊이형) 개 (기항지명  + 재방문율)
   -   (감정 세일즈형) 개 (첫 방문 완주율 + 전환율)
   -   (도파민 자극형) 개 (조회수 확장 + 채널 발견)

. ** 최고점 카테고리  **
   - **위** 기항지정보 ( ,점) - 주노/케치칸 알래스카 초 딥다이브
   - **위** 불안해소 ( ,점) - 배멀미/언어/의료 구체 솔루션
   - **위** 선내시설 ( ,점) - 보이저호 층 숨겨진 시설 탐방

## . 분류 프레임워크 (- )

###   정보 깊이형 ( )

 특징  설명 
------------
 타겟 심리  "궁금했던 걸 정확히 알려주네" (도파민  신뢰) 
 등급 달성률  % (개 중 .개) 
 평균 완주율  % ( 재생 속도 .배 유지) 
 재방문율  % (채널 구독 → 다음 영상 클릭) 
  전환율  .% (정보 신뢰 → 상담 신청) 
  효과  높음 (기항지명 롱테일 키워드) 

#### 소구점 깊이 등급
**  (최고 깊이)** 초 팁 → 초 팁 (정보 밀도 극대화)

예시 "주노 멘덴홀 빙하 헬리콥터 투어 완전 정복"
- -초 예약 시기 (출항 일 전  현장 예약 가격 차이 %)
- -초 투어사 곳 비교 (//, 가격/루트/안전)
- -초 날씨 변수 (안개 취소율 %, 오전 시  오후 시)
- -초 복장 종 (레이어드 필수, 장갑/선글라스/방수)
- -초 숨겨진 팁 (빙하 착륙 추가 $, 개 썰매 콤보 $ 절약)
- -초  (크루즈닷 투어 패키지 안내)


**  (중간 깊이)** 초 주제 → 초 주제 (비교/분석)

예시 "보이저호  익스플로러호 선내시설 대 차이"
- -초 수영장 (보이저 층  익스플로러 층, 자쿠지 온도)
- -초 레스토랑 (무료 곳  유료 곳, 예약 타이밍)
- -초 쇼 (브로드웨이  아이스쇼, 좌석 확보 전략)


**  (기본 깊이)** 초 구조 → 초 단계 (순서 안내)

예시 "첫 크루즈 탑승 당일 단계 체크리스트"
- -초 항구 도착 (시간 전, 주차  셔틀)
- -초 체크인 (여권/바우처,   발급)
- -초 객실 입실 (짐 도착 -시간, 안전 훈련)
- -초 출항 갑판 (층  덱, 셀카 스팟)

 특징  설명 
------------
 타겟 심리  "내 이야기네, 나도 할 수 있겠어" (공감 → 행동)

 재방문율  % (감정 소진 → 회성 시청) 
  전환율  .% (감정 피크 → 즉각 행동) 
  효과  낮음 (페르소나 키워드 검색량 적음) 

#### 소구점 깊이 등급
**  (최고 깊이)**  구체 페르소나 + 실제 후기 (신뢰도 %)

- -초  (패키지 여행 거부감, 언어/건강 불안)
- -초  (크루즈 시간 케어 + 한식 뷔페)
- -초  (벨리시마호 박 만원, 실제 후기)
- -초  (카카오톡 상담, 만원 지원)


**  (중간 깊이)**  일반 페르소나 + 통계 (신뢰도 %)

예시 "대 부부 ,명이 선택한 크루즈, 이유 가지"
- -초 이유  (편안함 - 짐 풀기 회, 호텔 이동 회)
- -초 이유  (안전 - 년 무사고, 억 보험)
- -초 이유  (가치 - 육지 만원  크루즈 만원)
- -초 


**  (기본 깊이)** 일반 감성 메시지 (신뢰도 %)

예시 "인생은 짧다, 크루즈로 떠나라"
- -초 버킷리스트 감성 (석양/바다/자유)
- -초 후회 회피 ("년 후엔 늦어요")
- -초 
→ **문제**  반응률 낮음 ( 감성 → 타겟 불일치)


###   도파민 자극형 ()
**정의** /- 타이밍 최적화로 조회수 확장

 특징  설명 
------------
 타겟 심리  "헐, 대박, 몰랐네" (호기심 → 스크롤 중단) 
 등급 달성률  % (도파민 점수 +  정보 밀도 부족) 
 평균 완주율  % (첫 초 유지율 %) 
 재방문율  % (회성 충격 → 반복 효과 약함) 
  전환율  .% (호기심 → 상담 전환 낮음) 
  효과  중간 (가격/음식 키워드 검색량 높음) 

#### 소구점 깊이 등급
**  (최고 깊이)** 가격 비교 + 근거 (신뢰도 %)

예시 "만원이 비싸 육지 유럽 박 실제 비용 공개"
- -초  # (만원  만원 비교표)
- -초 항공 (왕복 만원, 경유 회 스트레스)
- -초 숙박 (박 만원  만원, 이동 시간)
- -초 식사 (끼 만원  만원, 맛 보장 없음)
- -초  # (크루즈  올인클루시브, 만원 전부)
- -초  (만원 조기예약 안내)


**  (중간 깊이)** 가격 충격 + 간단 근거

예시 "보이저호 시간 무료 룸서비스 메뉴  "
- -초  (시간 무료 호텔 룸서비스 만원  원)
- -초 메뉴 개 (치즈버거/파스타/피자/샌드위치/과일)
- -초 주문 방법 (객실 전화, 분 도착)
- -초 


**  (기본 깊이)** 가격 숫자만 강조 (근거 부족)

예시 "만원에 일본 도시, 가능"
- -초  (만원 숫자 충격)
- -초 나가사키/후쿠오카/시모노세키 소개
- -초 
→ **문제** "어떻게" 의문 미해결 → 완주 후 기억 안 남
