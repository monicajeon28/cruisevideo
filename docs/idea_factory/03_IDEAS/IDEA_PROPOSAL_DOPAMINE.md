# 도파민 최적화 아이디어 제안서
##   + - 타이밍 전략

**작성일** --
**작성자**  - 상위 % 퍼포먼스 마케터 (도파민 최적화 전문가)
**목표** 완주율 % 달성 (현재 % → +%)

---

##  

### 핵심 전략 - + -- 타이밍 최적화


타이밍 전략 (   극복)
┌────────────────────────────────────────────────────────┐
│ 초   -    충격적 가격/수치 (도파민 차 주입) │
│ 초  - -   반전 정보 ( 탈출)           │
│ 초  -   한정성 메시지 (긴급성 각인)       │
│ 초  - -   마지막 기회 (클라이맥스 예고)     │
│ 초  -    행동 촉구 ( 직전 푸시)         │
└────────────────────────────────────────────────────────┘


### 도파민 트리거 최적화 원칙

. **숫자 우선** 모든 에 구체적 숫자 포함 (만원, 석, 가지)
. **초단타 메시지** -글자 (.-.초 읽기 시간)
. **시각 자극**  이미지 매핑 (텍스트와 연계된 비주얼)
. **반복 리듬** 초 간격 (→→→초는 , 불균형 리듬이 )
