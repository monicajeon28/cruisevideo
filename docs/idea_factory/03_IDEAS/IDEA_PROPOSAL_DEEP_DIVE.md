# 크루즈 콘텐츠 정보 깊이 템플릿 — 초 세부 소구점 설계서

 **작성일** --
 **역할** 상위 % 여행 콘텐츠 전문가 (정보 깊이 설계)
 **미션** "세부 주제 하나를 초 안에 깊게 파고들기" 템플릿 설계
 **입력**   완료 후 .
 **출력** 카테고리별 세부 소구점 템플릿 개 + 예상 등급 점수

---

## . 정보 깊이 프레임워크 원칙

### 핵심 원칙 가지

 원칙  설명  예시 
------------------
 **구체성**  추상적 형용사 대신 측정 가능한 수치/위치/시간대 제시   "경치가 좋아요" →  "층 발코니에서 새벽 시 분 일출 정면" 
 **검증가능성**  누구나 재현/확인 가능한 정보만 포함   "최고의 투어" →  "주노항 분거리,  헬기투어 평점 ./ (리뷰 ,개)" 
 **자 제약**  등급 기준 —  문장 최대 자 ( .  초)  기항지 개 + 가격 개 +  개  정보밀도 % 

### 초 정보 배치 타임라인


초   세부 주제 즉각 공개 (숫자/위치/시간대 포함)
초   핵심 정보 # (가장 중요한 )
초   핵심 정보 # (비교/대안/타이밍)
초   핵심 정보 # (실행 가이드/팁)
초  카카오톡 링크 클릭 유도 +  요소
