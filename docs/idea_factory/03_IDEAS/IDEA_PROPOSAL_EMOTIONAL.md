#  감정 세일즈 템플릿 종 - 크루즈   전용

**작성일** --
**담당자**  상위 % 감정 설계 세일즈 전문가
**목적** 초 영상에  단계 감정 여정 매핑
**타겟** 한국  세대 (크루즈 여행 주 고객층)

---

## .  프레임워크 개요

### .  단계 정의


 -  (문제)     -초       공감 유도 + 호기심
 -  (동요)   -초  -  불안/욕망 증폭
 -  (해결)    -초  -  해결책 제시 + 가치 증명
 -  (제안)       -초  -   구체적 혜택 제안
 -  (한정)   -초       긴급성/희소성
 -  (행동)      -초  -   명확한 행동 지시


### . 감정 타이밍 최적화 (초 구조)

 시간  단계  감정 상태   전략   메시지  목표 
-----------------------------------------------------
 -초    호기심 + 동경  선박 외관 웅장함  (없음)  시선 고정 
 초     **충격 + 긴급성**  숫자 강조  "월 만원"  첫 각인 
 -초   -  안전감 + 편안함  선박 내부 시설  (없음)  신뢰 구축 
 -초     설렘 + 기대감  기항지 명소  (없음)  욕망 증폭 
 초     **가치 인식**  포괄성 강조  "숙박+식사 전부"  중간 재고정 
 -초   -  확신 + 소속감  실제 후기/사람들  (없음)  결정 확신 
 -초  -  경이로움 + 만족  감정 정점  (없음)  감정 강화 
 -초   -   + 행동욕구   텍스트 강조  (없음)  행동 준비 
 초     **신뢰 확인**  사회적 증거  "천 가족 다녀옴"  최종 확신 
 -초     신뢰 + 확신   요소  (없음)  행동 촉발

## . 감정 세일즈 핵심 원리

### . 감정  구매 결정의 %

**감정 트리거 공식**

   감정 트리거 단어 + 구체적 이미지 + 선박/기항지명


**예시**


  "" " 벨리시마 성급 선실에서 알래스카 빙하 보며 아침 먹어요",
  "" 
    " 벨리시마",     // 신뢰 트리거 (브랜드)
    "성급",            // 욕망 트리거 (럭셔리)
    "선실",             // 안전감 트리거 (공간)
    "알래스카",          // 동경 트리거 (목적지)
    "빙하",             // 설렘 트리거 (경험)
    "아침"              // 구체성 트리거 (시간)
  ,
  "" "동경 + 안전감 + 설렘"



### . 감정 키워드  뇌 반응 매핑

 키워드 유형  감정  뇌 화학물질  효과  예시 
------------------------------------------
 럭셔리  욕망 + 동경  도파민 +%  "갖고 싶다"  "성급", "프리미엄" 
 안전  신뢰 + 안정  옥시토신 +%  "믿을 수 있다"  "년 경력", "억 보험" 
 희소성   + 긴급  코르티솔 +%  "지금 안 하면"  "잔여 명", "마감 임박" 
 혜택  이득 + 만족  도파민 +%  "득템했다"  "만원 할인" 
 포괄성  편안함 + 안심  세로토닌 +%  "걱정 없다"  "숙박+식사 전부"

## .  템플릿 종

### 카테고리 분류 (. 기반)

- ** 핵심 개** ( .) 불안해소, 기항지정보, 선내시설, 여행기준교육, 프리미엄경험
- ** 가치 강조 개** ( .) 음식, 가격가성비, 꿀팁, 비교, 럭셔리
- ** 추가 개** ( .) 버킷리스트, 시리즈, 시니어, 후기리뷰, 의료안전, 가족여행, 예약가이드, 동반자설득, 예약후안심, 짐싸기준비
