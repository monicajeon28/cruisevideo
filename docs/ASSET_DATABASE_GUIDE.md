#  사용 가이드

## 개요

  기반 로컬 에셋 데이터베이스 시스템

### 주요 기능

-  **,+ 에셋 확장 지원**
-  **전문 검색 ()** -  유사도 스코어링
-  **자동 메타데이터 추출** -  사용
-  **중복 감지** - 파일 해시 기반 ()
-  **사용 통계** - 페르소나별 선호도 분석
-  **성능 최적화** - 인덱싱, 캐싱,

##  레퍼런스

###  클래스

#### 에셋 추가


  .(
    '///.',
    '',          # '', '', '', ''
    '', '',
    '',
    '',         # 쉼표 구분 가능 ','
    .,                 # 초 단위
    '',
    '' '',   # 추가 정보
         # 중복 체크 스킵
)


#### 검색


# 전문 검색 ( )
  .(
    '', '',
    '',
    '',
    '',
    ,
      #  단순  검색
)

# 페르소나별 필터
  .('', )

# 카테고리별 필터
  .('', )

# 로 조회
  .()

# 해시로 조회 (중복 감지)

  .(
    ,
    '', '', '',
    ''
)

# 삭제
  .()

# 사용 기록
.(
    ,
    '',
    ''
)

# 인기 에셋 조회
  .('', )

# 페르소나 선호도 분석
  .('')
("선호 카테고리 ''")
("평균 재생 시간 ''초")


#### 디렉토리 스캔


  .(
    '//',
    ,           #  자동 감지
    ,     # 확장자로 타입 감지
    '',     # 기본 페르소나
    '',         # 기본 카테고리
          #  메타데이터 추출
)

("추가 '', 스킵 '', 에러 ''")


#### 최적화 & 내보내기


#  최적화 (, )
.()

#  내보내기
.('.', '')

#  가져오기
  .('.')

## 

** 기존 와 차이는**

 는  를 사용하여 확장성과 검색 성능이 향상되었습니다. 기존 는 완벽 호환됩니다.

**  모드와  모드 차이는**


-  ,개 에셋, 메모리 로드, 단순 검색
-  ,+ 에셋, 디스크 기반,  전문 검색

** 마이그레이션 후  파일 삭제 가능**

 보관 권장. 백업 용도로 사용하거나 레거시 모드 에 필요.

** 성능이 느려지면**


. .() 실행
. 인덱스 확인
. 캐시  조정
.  크기 확인 ( 초과 시 샤딩 고려)

** 다른 프로젝트에서 사용 가능**

 가능. .와 .를 복사하고 경로만 수정하면 됩니다.
