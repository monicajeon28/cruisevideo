-   **자  .자/초  초** (목표 초의 배)

**결과**
- 템플릿 자 →  검증 → 점 미만 → "" 반환
- .는 카테고리만 변경하고 **다시 자 템플릿 생성**
- 무한 루프 (재생성 아님, 카테고리 변경만)
