### 전략적 전환
- **현재**  단일 페르소나 (월 명 유입)
- **목표** 대 개 페르소나 (월 명 유입, +%)
- **브랜드 포지셔닝** "대접받는", "스마트한", "안전한", "함께하는" 크루즈

 지표  현재 ( 단일)  목표 (페르소나)  증가율

- 대/대/대/대 개 페르소나 전략 채택
-   개 공통,   페르소나별 차별화

**결과**
- 장점
  - 타겟 모수 +%
  - 월 유입 +%
  - 다양한 상품 포트폴리오 커버

  - 제작 리소스 배 증가
  - 브랜딩 일관성 관리 복잡도 증가
  - 초기 시행착오 리스크

**대안**
- 개 페르소나 (대+대) 리소스 절반, 효과 %
