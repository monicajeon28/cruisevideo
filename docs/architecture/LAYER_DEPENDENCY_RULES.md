## 체크리스트 의존성 규칙 준수

-   ** **

  -    의존성 없음
  -   순수 비즈니스 로직만 포함

-   ** **
  -    타입만 사용 (구현체 직접 사용 금지)
  -    예외만 처리 ( 예외 직접 처리 금지)

-   ** **
  -    구현만 수행
  -     의존 금지 (레이어 건너뛰기 금지)
