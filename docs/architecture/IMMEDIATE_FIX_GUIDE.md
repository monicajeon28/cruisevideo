##  성공 기준

### 필수 기준 (% 충족 필요)
-   총 글자 수 ≤ 자
-   예상   ≤ 초
-   등급 점수 ≥ 점
-     정확히 개 (///초)
-   - 최소 개 (/초 권장)
-    단계 구조 (++)
-    요소 최소 개 (년 경력, 억 보험 등)

### 선택 기준 (% 이상 충족 권장)
-    키워드 회 이상 언급
-     개 (싸다, 저렴, 완벽 등)
-    점수 점 이상 (구체적 숫자 사용)
-     점 이상 (가격+일정+혜택)

### 비즈니스 기준 (주일 후 측정)
-   등급 통과율 ≥ % (시간 이동 평균)
-   평균 생성 시간 ≤ 분
-   평균 재시도 횟수 ≤ .회
-   월 영상 생산량 ≥ 편
