## 위험도 평가

 항목  현재 상태  목표 상태    비즈니스 영향 
------------------------------------------------
  크루즈 비주얼  % (기항지 % 섞임)  %  -%  선박 판매 전환 -% 
  기항지 비주얼  % (크루즈 % 섞임)  %  -%  여행지 어필 -% 
  균형 비주얼  / (랜덤)  / (의도적)   (우연)  혼합형은 운 좋게 정상 
 등급 검증 활용률  % (버려짐)  %  -%  품질 관리 무용지물 

**총 예상 손실**
- 영상당  .% → .% (-%)
- 월 조회수 만 → 만 (-%)
- 월 상담 전환 건 → 건 (-%)
- **월 매출 손실 약 만원** (건만원/건)
