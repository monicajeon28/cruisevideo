#   - 에셋 매칭 테스트 가이드

**목표** 영상 렌더링 **없이**  에셋 매칭 로직 검증

**예상 소요 시간** 분 (영상 생성 대비 % 단축)

---

## . 테스트 개요

### . 검증 포인트 ()

   검증 항목  예상 결과 
--------------------------
 -   감지  // 정확 분류 
 -   적용  /일 때 +점 부스트 
 -  에셋 선택 비율   → 기항지 %,  → 크루즈 % 
 -  점수 우선순위   적용 에셋이 더 높은 점수 
 -  에셋 파일 존재  선택된 에셋이 실제로 로컬에 존재 

### . 사용 가능한 로컬 에셋



├── 
│   ├── 크루즈정보사진정리   (크루즈/기항지 사진)
│   ├── 후기                 (후기 이미지,  우선)
│   └── 로고                 (크루즈닷 로고)
├── 
│   ├──                  ( 전용 영상 개)
│   └── (기타 크루즈 영상)
├──                   (효과음 종)
├──                     ()
└──       ( 생성 이미지)


### . 테스트 대상 스크립트

 스크립트  예상   이유 
---------------------------
 .    산토리니/두브로브니크/스플릿 등 기항지 개 
 .    크루즈 올인클루시브 + 기항지 균형 
 .     벨리시마 선박 어필
