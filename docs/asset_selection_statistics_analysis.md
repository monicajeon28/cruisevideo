#     ( )

## 목적
 가중치가 실제로 에셋 선택에 영향을 주는지 정량 검증

## 배경
- ** **에서 --- 구현 완료
-  (//) 감지 →  전달 →  적용
- **검증 필요** 가중치가 실제 선택 비율에 영향을 미치는가

## 예상 결과
    설명  예상 크루즈 비율  예상 기항지 비율 
--------------------------------------------------------
   크루즈 전용 ( 벨리시마, 로얄캐리비안)  %  % 
   크루즈+기항지 복합  %  % 
   기항지 전용 (산토리니, 나가사키)  %  % 

## 통계 수집 항목

### . 에셋 카테고리 분류
각 선택된 에셋을 다음 카테고리로 분류
- **크루즈 에셋** 파일명/경로에 "", "벨리시마", " ", "선박" 포함
- **기항지 에셋** 파일명/경로에 "산토리니", "두브로브니크", "나가사키" 등 기항지명 포함
- **기타 에셋** 위 두 카테고리 외 (예 "후기", "" 등)

### .  적용 여부
-  로그에서      빈도 수집
- /에서 해당 카테고리 에셋의  적용률 측정

### . 최종 선택 에셋  
- 각 스크립트에서 최종 선택된 에셋 개 리스트
- 점수 ( +  +  + ****)
- 카테고리 (크루즈/기항지/기타)

## 검증 기준

###  조건
- **** 크루즈 비율  %
- **** 기항지 비율  %
- **** 크루즈/기항지 비율 차이  %
-  적용 시 해당 카테고리 평균 점수  다른 카테고리 +점

###  조건
- 위 조건 미달 시
-  적용되지 않음 (적용 횟수  )
- / 비율이 랜덤과 차이 없음 (와 유사)

## 데이터 수집 방법

### 방법  로그 파싱 (추천)
- 파이프라인 로그 파일에서 정규식 패턴 추출
- 패턴      (, , )
- 빠르고 정확, 파이프라인 수정 불필요

### 방법  파이프라인 수정 (선택)
- .에 통계 수집 코드 추가
-  리스트 분석
-  형식으로 저장 (//.)
- 더 상세한 정보 (, 키워드 등) 수집 가능

## 예상 

### 검증 성공 시
-  가중치가 정상 작동 확인
-  영상 크루즈 선박 비주얼 % → 선박 판매 전환 +%
-  영상 기항지 비주얼 % → 여행지 어필 전환 +%
- 시각적 메시지 일관성 +%

### 검증 실패 시
-  로직 재검토 필요
-  파라미터 전달 오류 가능성
- 에셋 키워드 매핑 보강 필요

## 테스트 케이스

### 테스트   (크루즈 전용)
- **스크립트** . ( 벨리시마 특화)
- **예상** 크루즈 %, 기항지 %
- **** 크루즈 에셋 회 적용

### 테스트   (기항지 전용)
- **스크립트** . (산토리니, 두브로브니크 특화)
- **예상** 기항지 %, 크루즈 %
- **** 기항지 에셋 회 적용

### 테스트   (균형형)
- **스크립트** . (크루즈+기항지 복합)
- **예상** 크루즈 %, 기항지 %
- **** 회 (는 부스트 없음)

## 다음 단계

###   검증 완료 후
. 통계 결과를  .에 반영
.  값 튜닝 ( →  혹은 유지)
. 별 에셋 풀 확장 (  기항지 이미지 수집)
. 와 연계 (비주얼 일관성 점수)

###   착수 시
- .에서  수동 선택  추가
- .에서  자동 감지 + 가중치 적용
- 에셋 선택 로그 실시간 대시보드 (선택)
