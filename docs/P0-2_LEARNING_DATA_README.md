# -  프롬프트 주입

##  개요

**작업 기간** 일
**우선순위**  (최우선)
**상태**  **완료**

  프롬프트에 // 개  섹션을 주입하여 **등급 콘텐츠 품질 기준을 실제로 적용**.

기존에는 섹션 파일이 존재했지만 사용되지 않았음. 이제 자동으로 로드되어  프롬프트에 주입됨.

---

##  문제점

###  (주입 전)

# . - 

 ()
    #  하드코딩된 프롬프트만 사용
      """
    당신은 크루즈닷 브랜드 전문 컨텐츠 작가입니다.
    ... (기본 프롬프트만)
    """

      ..()


**결과**
-   섹션 개 파일 **활용 %**
-  등급 품질 기준 **미적용**
-   ,   등 **검증되지 않음**
-  일관성 없는 콘텐츠 품질

##  등급 콘텐츠 품질 기준 ()
**활성화된 섹션** 개

###    - -초 최적화 ()
**등급 목표**  % 이상,  ./ 이상

**핵심 원칙**
- 부정 명령형 - 즉각적 주의 환기 (절대 하지 마세요 / 이것만 모르면)
- 내부자 비밀 - 정보 격차 유발 (직원만 아는 / 승무원이 말 못 하는)
- 충격적인 숫자 - 인지 부조화 유발 (예상과 다른 가격)

**예시 ()**
  - "이것만 모르면 크루즈 여행 망칩니다"
  - "크루즈 타기 전 절대 하지 마세요"

**필수 요소**
    첫 초에 강력한 패턴 인터럽트 존재
    안녕하세요/여러분/오늘은 단어 제거
    구체적 숫자 또는 시각적 임팩트 포함

##  성과 지표

###   

 항목      개선율  측정 가능 
---------------------------------------
  활용률  %  %  +∞   측정 완료 
 등급 기준 적용      -   코드 검증 
  품질 점수 (측정 가능)  점 (예시)  -점 (목표)  +% (이론적)    
 초 리텐션  % (추정)  % (목표)  +% (이론적)     필요 
 콘텐츠 일관성  낮음  높음  -   정성적 평가 

**측정 근거**
- ** 품질 점수** .() 메서드로 실시간 측정
  -  예시 "안녕하세요 여러분, 오늘은..." → 점 (금지 패턴 포함)
  -  목표 "이것만 모르면 크루즈 만원..." → 점 (등급 기준 충족)
  - 실제 측정  - "    (().('텍스트'))"

- **초 리텐션**  **현재 측정 불가** (   미연동)
  - % → %는 **이론적 목표**이며 실제 데이터 아님
  - 대체 지표  품질 점수를 대리 지표로 사용 ( 점 이상 → 리텐션 % 예상)
  - 실제 측정을 위해서는    연동 필요

**참고 문서** /. - 전체 측정 기준 및 근거
