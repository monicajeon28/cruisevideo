#   가속 자동 편집 시스템 사용 가이드

## 개요

**목표** 영상 편집 속도 배 향상 (분 → .분)

  는 , ,  를 활용한 고속 영상 편집 시스템입니다.

### 주요 기능

. ** 가속 인코딩**
   -  . ( )
   - . ( )
   -  ()
   -   (자동)

. **자막 자동 배치**
   -    타이밍 (-, -)
   -   위치 강조 (% 지점)
   - 노란색 배경, 검은색 굵은 글씨, 
   - 화면 하단 % ( 최적화)

. ** 자동 믹싱**
   - 볼륨 자동 조절 (-  -)
   -  / (처음 초, 마지막 초)
   - 나레이션 구간 덕킹 (-)

. **성능 최적화**
   - 멀티스레드 인코딩 ()
   - - 인코딩 옵션
   - 메모리 최적화

## 성능 최적화

###  품질 설정

   품질  파일 크기  사용 사례 
-------------------------------
    최고  매우 큼  프리미엄 콘텐츠 
    높음  적당  **권장 기본값** 
    중간  작음  테스트/프리뷰 

###  설정

   속도  품질   영향 
------------------------------
   매우 빠름  낮음  작음 
   빠름  중간  작음 
   중간  높음  **권장** 
   느림  매우 높음  큼
