## 트러블슈팅

### 문제 "  " 에러

**원인** 에 이미지가 없거나 품질 기준을 충족하는 이미지가 없음

**해결**

# .  확인
 /. --

# . 이미지 재스캔
 /.

# . 품질 기준 낮춤 (임시)
# .에서 를 .으로 조정


### 문제 품질 점수가 모두 낮음

**원인** 이미지 품질이 실제로 낮거나, 분석 알고리즘 문제

**해결**

# 고품질 이미지 수동 선별
 /.


### 문제 같은 이미지만 반복 선택

**원인** 사용 가능한 이미지가 적음

**해결**

# 사용 이력 초기화 (테스트 목적)
 /. "  "

# 또는  조정 (.)
