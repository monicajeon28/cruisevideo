### 핵심 문제 가지

#### --  메시지 개 → 개 (+점)
**원인**
-  프롬프트에서  메시지 생성이 선택사항으로 처리됨
-  배열이 빈 배열로 반환됨

**해결**
-  프롬프트에  마커로 필수 지정
- 개 정확한 타이밍 지정 초/초/초
-  로직 가 누락해도 자동 보정

**예상 효과** +점 (/ → /)

#### --  단계 불완전 → 완전 (+점)
**원인**
- 에서 긴급성() 단계 누락
- /만 있고 단계 구조 미완성

**해결**
-  단계 명확히 지정 (.) → (.) → (.)
- 각 단계별 템플릿 제공
- 순서 및  검증

**예상 효과** +점 (/ → /)
