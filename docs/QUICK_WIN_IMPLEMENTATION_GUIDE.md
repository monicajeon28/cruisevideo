## 구현 순서 (우선순위)

###    개 강제 (분, 최고 )
- **이유**  만원/ (최고), 도파민 피로 즉시 제거
- **리스크** 낮음 ( 로직 있음)
- **검증** 즉시 테스트 가능

###   기항지 비주얼 (분,  만원/)
- **이유** 인지 부조화 제거 (가장 큰 이탈 원인 -%)
- **리스크** 중간 ( 수정 → 전체 영상 영향)
- **검증** 테스트 스크립트로 즉시 확인

###    타이밍 (분,  만원/)
- **이유**  심리 최적화
- **리스크** 낮음 (템플릿 수정만)
- **검증** 타이밍 계산 스크립트로 확인
