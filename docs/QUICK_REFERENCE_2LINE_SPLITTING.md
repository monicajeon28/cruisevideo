# 자동으로 줄로 분할됨!


### 동작 방식
- **자 미만** 줄 그대로
- **자** 의미 단위 줄 분할
- **자 이상** 강제 줄 분할

## 분할 우선순위

 우선순위  패턴  예시 
---------------------
   쉼표 (,)  "크루즈, 만원" → "크루즈," + "만원" 
   가격  "여행 만원" → "여행" + "만원" 
   조사  "부산에서 출발" → "부산에서" + "출발" 
   접속사  "크루즈 그리고 호텔" → "크루즈" + "그리고 호텔" 
   띄어쓰기  "산토리니 두브로브니크" → "산토리니" + "두브로브니크" 
   글자 수  (띄어쓰기 없으면 중간 분할) 

## 지원하는 조사 (개)


로, 은, 는, 가, 이, 의, 를, 을, 에서, 처럼, 마저, 조차


## 예시

###  쉼표 기준

  "산토리니 두브로브니크 박일 크루즈, 만원"
 "산토리니 두브로브니크 박일 크루즈,"
        "만원"


###  조사 기준

  "육지 여행으로 산토리니 두브로브니크 가려면"
 "육지 여행으로"
        "산토리니 두브로브니크 가려면"


###  가격 기준

  "월 만원  숙박+식사+이동 전부"
 "월 만원"
        " 숙박+식사+이동 전부"


###  짧은 자막 (분할 안 함)

  "크루즈닷"
 "크루즈닷" (줄)

## 성능

- **렌더링 시간** 초 (변화 없음)
- **메모리** +.% (무시 가능)
- **파일 크기**   (정상)

## 특징

-  자동 적용 (별도 설정 불필요)
-  의미 단위 분할 ( 수준)
-  가격 하이라이트 유지
-  그라데이션 배경 유지
-  % 하위 호환
