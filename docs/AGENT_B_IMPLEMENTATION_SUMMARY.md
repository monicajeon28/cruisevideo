### . 의 불안정성

**발견** 는 프롬프트를 % 확률로 무시함
- 아무리 강조해도 - 세그먼트를 생략

**교훈**  응답에 의존하지 말고 후처리로 검증 및 수정

### .   코드의 무용성

**발견** ()가 이미 .에 구현되어 있었음
- 하지만 .와 연동 안됨
-    이 작업했다고 하지만 실제로는 사용 안됨

**교훈** 기존 코드가 있어도 연동 안되면 무용지물. 직접 구현 필요.
