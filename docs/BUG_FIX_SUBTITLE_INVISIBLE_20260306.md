### 시각적 검증
-  위치  하단 여백() 위에 정확히 배치

- 줄바꿈 자 초과 텍스트 자동 줄 분할

## 예상 효과

- **자막 가시성** % → % (완전 해결)
- **완주율** +% (자막 인식 가능)
- **가격 인지** "만원" 금색 하이라이트로 +% 기억률
- **디버깅 시간** -% (상세 로그로 추적 용이)
