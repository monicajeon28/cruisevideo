## 성공 지표

**  목표**
-  전환 성공 →  영상 개당 초 → 초
- 등급 기준 조정 →  합격률 % → % 예상
-   완료 →  결혼 테마 데이터 확보

**  목표**
- 하루 생산량 개 → **개**
- 테마 다양성 개 → **개** (크루즈 + 결혼 + 맛집)
- 등급 합격률 % → **%**
