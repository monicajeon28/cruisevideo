##  현재 시스템의 치명적 한계 (  발견)

### . 단일 템플릿 (  개뿐)
**문제** 매일 개 생산 시 같은 구조 반복 → 어뷰징 감지
**해결**  종 () +  변형  가지 조합

### .  연동 없음 ( 하드코딩)
**문제** .. 실제 상품과 불일치 → 허위 광고 법적 리스크
**해결**   실시간 연동 (개 활성 상품 자동 동기화)

### .  이미지 부족 (개  필요 개)
**문제** 등급  점수 점 (점 손실)
**해결**  개 긴급 생성 (완료 )

### . 사후 수정 없음 ( 회 생성으로 끝)
**문제** 등급 달성률 % (개 실패/개)
**해결**  구현 (재작성 루프)

### . 기항지 이미지 부족 (아시아 장 미확보)
**문제**   점 미달
**해결** 기항지 크롤링 (시간)

### . 단일 가격 프레임 ("싸다" 반복)
**문제** 고가 세일즈 포지셔닝 위배
**해결** "옳다" 프레임 +  티어 신설 (만+ 형)

### . 다양성 부족 (에셋 재사용 리스크)
**문제** 같은 이미지 반복 → 어뷰징 의심
**해결**   가지 변형 + 썸네일 개
