## .  시각적 품질 기준

### . 자막 가독성 ()

     (  공식 권장)
실제 여백
  - 폭 ( - ) /    (좌우 각 )
  - 높이 ( - ) /    (상하 각 )

폰트 크기  (  --)
줄당 글자수 자 (       -  여백)
최대 줄수 줄
배경 불투명도 . (대비  이상)

합격 기준
  - % 프레임에서  내 자막 포함
  - 줄 자막 높이   (   + 패딩 )
  - 노치 디바이스 시뮬레이션 통과 (  )


### .  메시지 가시성 (등급 필수)

타이밍 정확도 .초 (., ., .)
폰트 크기  (자막  대비 +%)
대비 배경  텍스트  이상
애니메이션 .초 (너무 빠르지 않음)
가시성  시니어 명 육안 테스트 % 읽기 가능

등급 채점
  -  개 정확 .점
  -  개 .점
  -  개 .점
  -  개 .점


### . 이미지 관련성 (등급 필수)

기항지 키워드 매칭
  -  개 (  -)
  - 기항지 포함 세그먼트 % 이상 해당 기항지 실사

매칭 점수
  - -점 완벽 (산토리니 → 산토리니 석양)
  - -점 양호 (산토리니 → 그리스 에게해)
  - -점 보통 (산토리니 → 지중해 일반)
  - 점 불합격 (산토리니 → 크루즈 선박)

등급 기준
  - 기항지 비주얼 ≥ 초 (전체 초의 %)
  - 기항지 키워드 ≥ 개 (스크립트 내)
