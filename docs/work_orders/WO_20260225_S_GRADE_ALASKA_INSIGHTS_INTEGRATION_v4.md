####  문제  -비주얼 문장 단위 불일치 (완주율 직격탄)
**현상**
-  "산토리니의 석양이 아름답습니다"
- 영상 수영장 사진 ()

**원인**
-  단위 비주얼 매칭 (초  개 비주얼)
- 문장별 키워드 추출 없음
-  이 /  점령

**영향**
- 완주율 % (목표 %)
- 시청자 인지부조화 → 즉시 이탈
- 도파민 % → 지루함

**알래스카 라이브방송 교훈**
- 시간 내내 **비주얼   키워드 % 일치**
- 화면 "만년 빙하" 말할 때 → 빙하 사진 즉시 전환
- 개 -비주얼 매핑 리스트 확보 ( )

####  문제   메시지 후킹력 % (목표 %)
**현상**
-  "매일 다른 나라" (밋밋함, 감정 자극 %)
-  이미지 일반 선박 사진 (후킹 %)

**원인**
- 심리학 기법 미적용 (손실 회피, 희소성, 대비 효과 등)
- 이미지 매칭 키워드만 사용,  없음

**영향**
-  점수 / (-점)
- 초 지점 이탈률 %

**알래스카 라이브방송 교훈**
-   **"패키지 만  우리 만"** (점, 대비 효과)
- **"만년 빙하 녹는다"** (점, 손실 회피 + 희소성)
- **"선착순 명"** (점, 희소성)
- 개 고후킹 문구 확보 ( )

**영향**

##  해결 전략 알래스카 인사이트 통합

### 전략  교육형 세일즈 (  + )

**라이브방송 공식**

 (분) → 가치 교육 (분) → 가격 공개 (분) →  증폭 (분)


** 초 압축 ( 압축률)**

 (초) 손실 회피 + 구체적 숫자
  ↓
 교육 (초) 가치 요소 + 신뢰 요소
  ↓
가격 공개 (초) 대비 효과 (육지  크루즈)
  ↓
 증폭 (초) 긴급성 + 행동 + 신뢰


**핵심 원칙**
- 가격은 **가치 이후** (순서 바꾸면 실패)
- 신뢰 요소 **가지 중 최소 가지** 포함
- - **회 필수** (초, 초 지점)
