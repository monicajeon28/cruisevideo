## . -  아키텍처

### .  개요

  로컬 에셋 ()
 ├─ 비용 $
 ├─ 속도 .초
 └─ 성공률 % (/ 기항지)
      ▼ (매칭 실패)

  /  (무료)
 ├─ 비용 $
 ├─ 속도 초
 ├─ 성공률 %
 └─ 제한   /,  무제한
      ▼ (매칭 실패)

   - (유료, 영상)
 ├─ 비용 $./초
 ├─ 속도 초
 └─ 성공률 %
      ▼ (영상 불필요 시)

     (유료, 이미지)
 ├─ 비용 $./장
 ├─ 속도 초
 └─ 성공률 %


### . 주요 함수

 (, )    검색   
 (, )    검색   
 (, , )   영상 생성

## . 기대 효과

### . 비주얼 품질
 지표      개선 
---------------------------
 기항지 매칭률  %  %  +% 
 등급 비주얼 점수  점  점  +점 
 에셋 다양성  개  개 + ∞  +% 

### . 비용
- 초기 투자 $. (회)
- 운영 비용 $./월 (편 기준)
-   % 달성 + 등급 보장

### . 생산성
- 에셋 수집 시간 → 초 (-%)
- 신규 기항지 수동 → 자동 (즉시)
- 에셋 누락 생성 불가 → 자동 대체 (%)
