**작성일**: 2026-03-09
**Phase**: B - S등급 이상 (98.8 → 완전체)
**목표**: 상위 1% 감정 설계 + 100만 조회수급 콘텐츠 + 영상 프로덕션 + 마케팅 퍼널

## 🎯 Phase B 비전

**현재 상태 (Phase A 완료):**
- S등급: 97.0/100 (90점 이상 달성)
- Pop Messages 타이밍: 정확화 완료
- CTA 3단계: 긴급성-행동-신뢰
- 금지어 필터링: 검증 가능 사실만 사용
- Pexels API Fallback: 로컬 에셋 부족 시 자동 다운로드

**Phase B 목표:**
1. **감정 설계**: 상위 1% 세일즈맨급 감정 흐름 (안심→공감→동경→확신)
2. **콘텐츠 품질**: 100만 조회수급 바이럴 콘텐츠 (공유/신뢰/문의 유발)
3. **영상 프로덕션**: 다양한 비주얼 + 애니메이션 + Ken Burns + 자연스러운 전환
4. **마케팅 퍼널**: 강력한 후킹 (어그로) + 완벽한 CTA (전환율 10배)
5. **크로스체크**: 비판적 검증 + 코드 품질 검수 + 누수/충돌 제거

## 📋 서브에이전트 팀 구성 (병렬 실행)

### Agent 1: Emotional Architect (감정 곡선 설계 전문가)

**역할**: 처음부터 끝까지 감정을 끌고가는 세일즈맨급 스크립트 설계

**작업 범위:**
1. 감정 곡선 분석 (현재 vs 목표)
   - 현재: 안심(0.40) → 공감(0.55) → 동경(0.75) → 확신(0.65)
   - 목표: 불안(0.30) → 호기심(0.70) → 열망(0.90) → 확신(0.95)

2. 감정 트리거 설계
   - Hook: 공포/호기심 (0-5초)
   - Problem: 불안/공감 (6-15초)
   - Solution: 안도/열망 (16-35초)
   - Trust: 확신/신뢰 (36-50초)

3. 감정 전환 타이밍
   - Re-Hook 2회 (13초, 32초)
   - Pop 3회 (15초, 32.5초, 46.5초)
   - 감정 피크: 동경(26-40초)

4. 구현 파일
   - `engines/comprehensive_script_generator.py` (감정 곡선 검증 강화)
   - `engines/emotion_curve_validator.py` (신규 생성)
   - `config/emotion_templates.json` (감정 패턴 템플릿)

**산출물:**
- `EMOTIONAL_ARCHITECTURE_REPORT.md` (감정 설계 보고서)
- `emotion_curve_validator.py` (감정 검증 엔진)
- 감정 템플릿 20종 (공포형, 호기심형, 열망형 등)

### Agent 2: Viral Content Strategist (100만 조회수 전략가)

**역할**: 공유하고 싶고 신뢰가 가는 100만 조회수급 콘텐츠 전략

**작업 범위:**
1. 바이럴 패턴 분석
   - 공유 유발 요소 (놀라움, 공감, 실용성)
   - 신뢰 구축 요소 (데이터, 사례, 전문성)
   - 문의 유발 요소 (긴급성, 희소성, 사회적 증거)

2. 100만 조회수 공식
   - Hook 강도: 3초 안에 멈추게 만들기
   - 완주율: 50% 이상 (현재 35% → 50%)
   - CTR: 15% 이상 (현재 4.5% → 15%)
   - 댓글 유발: 질문/논쟁/공감

3. 콘텐츠 타입별 최적화
   - 정보형: 놀라운 사실 + 데이터
   - 감정형: 공감 스토리 + 해결책
   - 교육형: 실용 팁 + 단계별 가이드
   - 비교형: 논쟁 유발 + 기준 제시

4. 구현 파일
   - `engines/viral_score_calculator.py` (바이럴 점수 계산)
   - `engines/shareability_optimizer.py` (공유성 최적화)
   - `config/viral_templates.json` (바이럴 패턴 템플릿)

**산출물:**
- `VIRAL_CONTENT_STRATEGY.md` (100만 조회수 전략서)
- `viral_score_calculator.py` (바이럴 점수 엔진)
- 바이럴 체크리스트 (30개 항목)

## 🛠️ Code Quality Guardian (상위 1% 개발자 검수)

**역할**: 최종 코드 품질 검수 (누수, 에러, 코드 스멜, 충돌)

**검수 항목:**
1. **메모리 관리**
   - 메모리 누수 (memory profiler)
   - 객체 참조 순환
   - 캐시 무한 증가

2. **예외 처리**
   - try-except 누락
   - 예외 메시지 불명확
   - 복구 전략 부재

3. **코드 스멜**
   - 중복 코드 (DRY 위반)
   - 긴 함수 (50줄 이상)
   - 복잡도 과다 (Cyclomatic Complexity > 10)
   - 매직 넘버

4. **동시성 이슈**
   - Race condition
   - Deadlock 가능성
   - 공유 자원 동기화

5. **성능 병목**
   - O(n²) 알고리즘
   - 불필요한 반복
   - I/O 블로킹

**검수 도구:**
- pylint (코드 스멜)
- mypy (타입 검사)
- memory_profiler (메모리)
- cProfile (성능)

**산출물:**
- `CODE_QUALITY_REPORT.md` (코드 품질 보고서)
- 수정 필요 파일 리스트 (P0/P1/P2)
