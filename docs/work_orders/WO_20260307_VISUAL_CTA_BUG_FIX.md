## . 검증 체크리스트

### - 검증 ( )

-   .  -에  필드 추가됨
-   프롬프트에  사용 예시 추가됨
-    생성 스크립트의 에  배열 존재
-   에 ,  포함됨
-   .() 결과  점 이상

### - 검증 ( )

-   .  -에  필드 추가됨
-   프롬프트에  단계 구조 (//) 자세히 설명됨
-    생성 스크립트의  에  필드 존재
-    순서  →  → 
-   .() 결과  점 이상

### 통합 검증

-     생성 스크립트 총점 점 이상
-    +  합계 점 이상
-    템플릿 엔진과  생성 스크립트 모두 정상 작동
