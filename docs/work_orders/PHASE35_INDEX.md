###    전문가 (분)

- **파일** /.

- **해결**  구현

###    최적화 전문가 (분)
- **문제**  점 → 등급 불안정
- **해결**  완화 ("지금" 허용)
- **파일** //.

- **문제** 통합 순서 불명확
- **해결** 순차 통합 (→→)
- **검증** 등급 점 % 달성
