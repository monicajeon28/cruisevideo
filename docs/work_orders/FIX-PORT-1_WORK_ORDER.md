# -- 작업지시서 기항지 비주얼 초 확보

## . 문제 정의

### 현상
- **현재** 기항지 비주얼 .초 (목표 초 대비 -.초, % 부족)
- **영향** 등급   / → 목표 / (-.점)
- **손실** 완주율 -%,  -%, 전환율 -%

### 근본 원인 (단계 분석)

#### 원인  대본 세그먼트 부족 (핵심 원인)
- **현재 구조** (개) + (개) + (개) + (개)  총 개 세그먼트
- **기항지 세그먼트**  개 중 평균 개만 기항지 관련
- **평균 세그먼트 시간** 초/개
- **결과** .개  초  **.초** (현재 값)
- **목표 달성 필요** 초  초/개  **.개** (최소 개 필요)

#### 원인  기항지 키워드 누락 (세그먼트 미인식)
- **** 개 기항지 등록 완료 ( )
- **** 개 기항지 확장 키워드 매핑 완료 ( )
- **문제** 대본 세그먼트에 가 기항지명을 포함하지 않는 경우 발생
  - 예시 "산토리니 석양" 언급 →  "", "" (기항지명 누락)
  - 결과 () 메서드가 해당 세그먼트를 기항지로 인식하지 못함

#### 원인  비기항지 세그먼트 과다 (선박 시설 편중)
- **크루즈 선박 세그먼트** 객실, 뷔페, 수영장, 갑판 등 선내 시설
- **비율 문제**  개 중 개가 선박 시설 → 기항지 세그먼트 개로 제한
- **이상적 비율**
  -  (크루즈 선박 중심) 크루즈 개, 기항지 개
  -  (복합형) 크루즈 개, 기항지 개
  -  (기항지 중심) 크루즈 개, 기항지 개
- **현재** 대부분  스크립트가 기항지 개로 부족 → **개 필요**

## . 결론 및 권장사항

### -. 핵심 요약
. **근본 원인** 기항지 세그먼트 .개 (목표 .개 대비 -.개 부족)
. **해결 방안**
   -   (분)  종 템플릿 수동 수정 → 초 달성 (+.점)
   - 설정 구현 (시간) . + . → 자동 검증 (+.점 전체)
. **예상 효과**
   - 등급 점 → 점 (점 돌파 )
   - 월 수익 만원 → 만원 (+만원)
   - 시간당  **만원/**

### -. 실행 권장 순서
.  **즉시 실행** (오늘, 분)   ( 종 수정)
.  **내일 오전** (시간) 설정 구현 (. + .)
.  **내일 오후** (분) 통합 테스트 (개 스크립트 일괄 생성)
. ⏳ **이번 주** ( ) -- + -- 연계 → **등급 점 달성**

### -. 성공 기준
-  기항지 비주얼 시간 초 이상 (현재 .초)
-    점수 ./ (현재 ./)
-  등급 총점 점 이상 (현재 .점)
-  기항지 세그먼트 개 이상 (현재 .개)
-  자동 생성 성공률 % 이상
