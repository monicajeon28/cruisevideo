### 핵심 문제
- 자막   미준수 → % 디바이스에서 잘림 → 이탈 +%
-  타이밍 부정확 (초 → 초 오차) → 등급 -.점
- 이미지 랜덤 매칭 → 관련성 % → 신뢰도 하락
- 기항지 비주얼 부족 (.초/초) → 등급 -.점

### 해결 방안
- - 검증 시스템 (- → - → 육안)
-    ( +  )
-  자동 검증 (  +  타이밍)
-  시각적 회귀 테스트 (/ 통합)

### 예상 효과
- 등급 점 → 점 (+점)
-  .% → % (+%)
- 전환율 .% → .% (+%)
- 월 수익 만원 → 만원 (+%)
