## . 현재 문제점

### . 핵심 이슈
 문제  예시  원인 
------------------
 **키워드 우선순위 없음**  "크루즈 처음이시죠" → "처음" 매칭  모든 키워드 동등 취급 
 **고유명사 미인식**  " 벨리시마" → "", "벨리시마" 분리  복합명사 처리 없음 
 **불용어 미필터링**  "정말 너무 좋았어요" → "정말", "너무" 추출   부재 
 **맥락 무시**  "바다가 무서우신 분" → 바다 풍경 매칭  부정 맥락 감지 없음 
 **에셋 매칭 범위**   이미지만 적용  **영상 세그먼트 미적용**

  정확 매칭 (  )
