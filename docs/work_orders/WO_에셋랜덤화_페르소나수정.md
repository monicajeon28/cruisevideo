## . 현재 문제 분석

### 문제  에셋 랜덤화 의심

**사용자 보고** "계속 똑같은 영상만 매칭된다"

**코드 분석 결과**
-  세트로 중복 방지  ( , )
- .() 사용  ( , )
-  파라미터 전달  ( )

**실제 로그 분석** (방금 생성 영상)

  -   ()
   객실 복도 . ( /)
   객실 복도 . ( /)
...


**근본 원인**
. **키워드 추출 실패** 로 전달되어 매칭 점수 점
. **  과다 사용** 카테고리 ""으로 빠짐
. **  사용** 로컬 에셋이 있는데도  다운로드

### 문제  페르소나  위치/표시

**사용자 요청**
- 자막 위에 표시 (현재 자막과 겹침)
- 초씩 번만 등장 (현재 전체 구간)
- 작은 크기 유지

**현재 설정** (.  -)

   ("", "")  # 자막과 겹침
   .  # % 크기

### . 에셋 랜덤화 강화

#### . 키워드 추출 개선 ( -)

**현재 문제**

  # 빈 리스트로 전달


**수정안**

# 나레이션에서 키워드 강제 추출
  ()
      '크루즈', '바다', '여행'


#### .   랜덤화 강화

**현재** 같은 카테고리에서 순차 선택
**수정안** 전체 에셋 풀에서 완전 랜덤 + 셔플


# . 또는  내
 
  (.("**/*."))
.()  # 매 영상마다 셔플
        .   
  .()    .()


#### . 세션별 에셋 풀 셔플

**위치**  또는 () 시작부


 ()
    """매 영상마다 에셋 풀 순서 셔플 (완전 랜덤화)"""
     (, '')
        .(.)
     (, '')
        .(.)


### . 페르소나  수정

#### .  설정 추가 (.)


# 간헐적 표시 모드
   
           # 번 등장
   .  # 각 초
   (., .)  # 초, 초 시점

# 위치 수정
   ("", "")  # 자막 위
   .  # % 크기 (작게)


#### . 파이프라인 로직 수정 ( -)

**현재** 모든 세그먼트에 페르소나 표시
**수정** 지정된 시점에만 초간 표시


 ..
    # 간헐적 모드 지정된 시점에만 표시
      ..
      ..

       
        #  시점에서  동안만 표시
          .(, )
          .()
        #  위치 자막 위 ( -  - )
          .. -  -  - 
          .((, ))
        .()
