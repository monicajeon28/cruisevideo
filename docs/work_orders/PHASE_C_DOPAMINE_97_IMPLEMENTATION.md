## . 검증 체크리스트

###  - 완료 후
-   재후킹 이벤트 로그 확인 (////초)
-    단계 텍스트 확인 (-/-/-초)
-   편집 리듬 로그 확인 ( .,  .)

###  - 완료 후
-     동기화 확인 (//초)
-    구간 시각 대비 확인 ( .)

###  - 완료 후
-    텍스트 .초 등장 확인
-    볼륨 . 확인
-   완주율  파일 생성 확인

### 전체 검증
-    자동 점수 **+/**
-   테스트 영상 편 생성 (  .)
-   / 테스트 준비 (.  .)
