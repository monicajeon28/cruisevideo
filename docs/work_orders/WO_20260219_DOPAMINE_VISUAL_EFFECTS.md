## .  시각 심리학 기반 도파민 설계

### .  세대 시각 반응 특성


노화에 따른 시각 변화
- 수정체 황변 파란색 인식 저하 → 노란/주황/빨강 선호
- 대비 감도 저하 고대비(  +) 필수
- 동체 추적 저하 빠른 움직임보다 "큼직한 변화" 효과적
- 주변시야 축소 화면 중앙 % 영역에 정보 집중
- 감정 반응 "향수" "안도" "기대"  "놀라움" "공포"


### . 도파민 트리거 매핑 ( 특화)

 시간  도파민 트리거  시각 요소  강도 
-----------------------------------
 .-.초  **시각 충격**  화면 전체 플래시 + 큰 텍스트 등장   
 .-.초  **호기심 갭**  핵심 키워드 개 -   
 .-.초  **정보 확인**   자막 + 페르소나 등장   
 -초  **안심 전환**  부드러운 장면 전환 + 따뜻한 색감   
 매 초  **시각 리셋**  배경 변화 +  텍스트   
 //초  **마이크로 텐션**  - + 화면 강조 효과   
 마지막 초  **행동 유도**   펄스 애니메이션

## . 테스트 체크리스트

-   색보정  호출 성공 확인
-      -초 시각 확인
-   자막 / 부드러움 확인
-    키워드가 의미 있는 단어인지 확인
-    키워드에 금지어 포함되지 않는지 확인
