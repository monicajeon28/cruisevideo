**분석 결과**
- ** (-초)**  적합 (흰색 플래시 .초, 노란 자막, 임팩트 )
- **- 초**  타이밍 부정확 (초 실제  초 이상 권장)
- **- 초**  적합 (초 배치,  방지)
- ** 메시지**  ./.개 (타이밍 오류로 .개 누락)

**  ** (웹서치 결과)
-   연구  -초가  (현재  적합)
-   논문 짧은 영상 중독성은 "  " (가변 보상)
  - 초, 초, 초 지점에 예측 불가능한 보상 (/-) 배치
-   연구  시니어는 도파민 반응 속도 -% → 타이밍 여유 필요
  - 권장 초→초, 초→초 (현재 초/초 → 부분 적합)

**완주율 예측 모델**
