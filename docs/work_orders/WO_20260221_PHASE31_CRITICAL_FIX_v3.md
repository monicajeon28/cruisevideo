##     긴급 이슈 (가지 치명적 문제)

###   하드코딩 유지보수 불가 ()
**증상**
- 개 카테고리 대본 생성 로직이 . 내 하드코딩
- 기항지/선박/카테고리 수정 시 코드 직접 수정 필수
- 확장성 제로 (신규 항로 추가 시 줄 복붙)

**영향**
- 개발 속도 저하 (카테고리 개 추가  시간)
- 휴먼 에러 위험 (오타, 로직 누락)
-  / 차단 ( 연동 불가)

**근본 원인**
-  에서 "일단 동작하는 코드 우선" 접근
- 설정과 로직 분리 실패
