##  

### 핵심 목표 (가지)

. **완주율 % → %** (도파민 % 유지)
. **비주얼  % → %** ( 에셋 자동 생성)
. **-비주얼 동기화 초 → .초** ( )

### 투자  

 항목  비용  기대 효과 
-----------------------
 초기 투자  $. + 일 개발 ()  완주율 +% 
 운영 비용  $./월  비주얼  % 유지 
 예상 수익 증가  -  **+만원/월** 
 투자 회수 기간  -  첫 달 즉시

## . 문제 정의 ( )

### . 대 치명적 문제

#### 문제  도파민 곡선 단절 (초 지점 이탈률 %)

**현상**
-  초 →  초 전환 시 감정 단절
- 시각 자극 부족 (초 에 장 정적 이미지)
- 도파민  → 스크롤 이탈

**근거**
-   초 지점 이탈률 %
- 완주율 % (기준 % 대비 -%)
-   분석 도파민 골짜기 초 지점 미구현

**영향**
- 월 수익 예상 -만원 (완주율 -%  만원)

#### 문제  -비주얼 타이밍 불일치 (초 오차)

**현상**

 "산토리니 석양을 감상하고..."
  .초 ─┬─ "산토리니" (음성)
  .초 ─┴─ 산토리니 이미지 등장 (초 지연)
         └─ 인지 부조화 발생


**근거**
- 현행  . 추정 (초 오차)
-   제안   (.초 정밀도)

**영향**
- 몰입도 -% (인지 부조화)
- 등급 비주얼 점수 -점

#### 문제  비주얼 에셋  % (/ 기항지 에셋 부재)

**현상**
 기항지  에셋 존재  대체 방법  품질 
------------------------------------
 두브로브니크   로컬  -  + 
 아르고스톨리   없음  랜덤 그리스 이미지  - 
 자킨토스   없음  일반 바다 이미지   

**근거**
-  개
- 로컬 에셋 폴더 개 (크루즈정보사진정리)
-  개 (%)

**영향**
- 등급 비주얼 점수 점 (기준 점 대비 -점)
- 완주율 -% (부정확한 비주얼)

## .   도파민 % 전략

### . 도파민 곡선 단계


도파민 레벨
   ├─────┐                          ┌──────┐
      │     │                          │      │
    │     │                    ┌────┘      │
      │     │                    │           │
    │     └────────┐          │            │
      │              │          │             │
    │              └─────────┘              │
      │                                        │
     └────────────────────────────────────────┘
                              

-초   →  (+)
  트리거 가격 충격 "만원" + 시각 충격 (골든파티)
  비주얼  전용 폴더 우선 (개)
    낮은 목소리 "팔십구만원에..."

-초   →  (-)
  트리거 공감 "여행 가고 싶은데 비싸서..."
  비주얼  부부 고민하는 표정 (후기 이미지 우선)
    차분한 목소리

-초   →  (+)
  트리거 해결책 "크루즈는 숙박+식사+이동 전부 포함"
  비주얼  뷔페 + 발코니 객실 + 공연장
    설명 +  감탄

-초   →  (-)
  트리거 신뢰 구축 "년 만팀 + 억 보험"
  비주얼 후기 이미지 (// 우선)
    신뢰감

-초   →  (+)
  트리거 긴급성 "잔여 명 마감 임박"
  비주얼 카카오 채널 링크 + 전화번호 워터마크
   단계  (긴급성 → 행동 → 혜택)


### . 여행 욕구 트리거 가지

 카테고리  트리거  구현 방법  파이프라인 적용 
---------------------------------------------
 **감각 몰입**  시각 ( 이미지)  -  고품질 우선     
 **감각 몰입**  청각 ( 파도)   / 우선    
 **감각 몰입**  후각 (언어 표현)   "바다 내음"   대본 수정 
 **감각 몰입**  미각 (음식 비주얼)  뷔페 이미지 우선    
 **감각 몰입**  촉각 (언어 표현)   "바람 느끼며"   대본 수정 
 **사회적 증거**  동년배 후기  후기 이미지 우선 ()   - 
 **사회적 증거**  숫자 증거 (,명)   인포그래픽   미구현 
 **사회적 증거**  전문가 (년)   세그먼트 필수   등급 채점 
 **가격 앵커**  분할 지불 (월 만원)    .    
 **가격 앵커**  비교 프레임 (육지  크루즈)       
 **긴급성**  한정 수량 (명)       -- 
 **긴급성**  시간 제한 (이번 주)   세그먼트   렌더링 미구현

## . 테스트 계획

### 테스트  -비주얼 동기화 정확도

**목표** .초 이내 % 달성

**방법**
. 샘플 편 생성 ( )
. 각 영상에서 개 키워드 타이밍 측정
.  타이밍  실제 비주얼 등장 시간 비교

**합격 기준**
- 개 키워드 중 개 이상 (%) .초 이내
- 평균 오차 .초 이하

### 테스트  비주얼 매칭률

**목표** % 이상

**방법**
. 희귀 기항지 개 선정 (아르고스톨리, 자킨토스 등)
. 각 기항지당 개 감정 변형 (, , )
. -  매칭 시도 (총 회)

**합격 기준**
- 회 중 회 이상 (%) 매칭 성공
-   비율 % 이상
-   비율 % 이상
-  + 비율 % 이하

### 테스트  완주율 (/ 테스트)

**목표** +% 이상 (% → %+)

**방법**
. 샘플  기존 시스템 (편)
. 샘플  신규 시스템 (편)
. 각 그룹 업로드 후 주간  추적

**합격 기준**
-  그룹 완주율 ≥  그룹 + %
- 예 % → ≥%
