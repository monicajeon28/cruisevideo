##    검증 체크리스트

### 대본 엔진 검증
-    에 "여행 욕구 자극 종" 명시됨
-   에 가격 언급 건 (감각 트리거만)
-   감각 묘사 종 (시각/청각/후각/촉각/미각) 개 이상
-   트리거 종 중 개 이상 사용
-   이면 안심(의료진/보험), 이면 인생샷()

### 이미지 매칭 검증
-   () 메서드 정상 작동
-   산토리니 석양  평범한 갑판 (점수 역전 확인)
-    파티 이미지 -점 페널티 적용
-   두브로브니크 → 산토리니 오매칭 수정 완료

### 영상 매칭 검증
-    폴더에 파티 영상 개
-   그리스 개 + 알래스카 개 확보 완료
-     영상 % (이미지 %)
-     영상 % (이미지 %)

### 스크립트 구조 검증
-     -초 구간 존재
-   - 초 초 존재
-   - 초 초 존재 ( 필수)
-   () 통과 ( 점 이상)

### 이중 타겟 검증
-    키워드 % (경력, 보험, 안전, 전문가)
-    키워드 % (핫플, , 경험, 활동)
-   톤앤매너 중립 ("미쳤어요" , "상상 이상" )
-   가격 프레임 가치 비교 ("만원  만원 가치")

이미지 키워드 매칭만, "가장 예쁜 사진" 선택 로직 없음
영상  파티 % →  거부
스크립트  초 → 이미 이탈 후
결과 "가고싶다" %, 초 이탈 %


###  (++ 완료)

후킹력 / (+%)
대본 "빙하 무너지는 소리", "석양 에메랄드빛" 감각 트리거 종
이미지  점, 랜드마크/석양 우선
영상  파티 %, 그리스/알래스카 확보
스크립트   초, - 초 이탈 차단
결과 "가고싶다" % (+%), 초 이탈 % (-%)
