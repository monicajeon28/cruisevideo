#### 문제 시나리오
. ** 구조**
   
   
       "" "",
       "" "정식 등록 여행사입니다. 억 여행자 보험 포함되어 있습니다. % 고객 성공했습니다.",  #  나레이션 전문
       "" "억 여행자 보험 포함"  # 자막용 요약
   
   

. **현재 처리**
   - () → .('', ...) 우선 선택
   -  전체 문장 (자) 선택됨
   - (, , ) 호출
   - 자  자(줄 )  **개 자막으로 분할**
     - 자막  "정식 등록 여행사입니다" (초)
     - 자막  "억 여행자 보험 포함되어" (초) ← **중복 발생!**
     - 자막  "% 고객 성공했습니다" (초)

. **의도된 처리**
   -  필드 사용 → "억 여행자 보험 포함" (자)
   - 자 이하 → **개 자막만 생성** (초)
   - 중복 없음

#### 근본 문제
-    나레이션용 (길고 상세함, 자)
-   화면 표시용 (짧고 요약, 자)
- **현재 코드는 용을 자막으로 사용 → 과다 분할 → 중복 문구 발생**

#### 문제 시나리오
. ** 데이터**
   
   
       "" "",
       "" "억 여행자 보험 포함되어 있습니다",

. **현재 타임라인**
   
   초         .초        초
   ----------------------
   일반 자막 "억 보험 포함"                      ← 전체 구간
                 "억 보험"                    ← ..초 (.초)
                      ↑
                   중복 표시!
   

. **시청자 시점**
   - .초 "억 보험 포함" (일반 자막, 하단)
   - ..초 "억 보험 포함" (하단) + **"억 보험" (중앙, )** ← **혼란!**
   - .초 "억 보험 포함" (하단)

#### 근본 문제
-  자막  도파민 강조 효과 (.초 짧게, 큰 폰트, 중앙)
- 일반 자막  전체 나레이션 표시 (전체 구간, 작은 폰트, 하단)
- **두 시스템이 독립적으로 작동 → 타이밍 조율 없음 → 중복**

##  예상 효과 측정

### 정량적 지표

 지표      개선 
---------------------------
 **자막 중복 발생률**  %  %  -% 
 **평균 자막 개수/영상**  개  개  -% 
 **자막 평균 길이**  자 → 개 분할  자 → 개  % 효율 
 **-자막 겹침 구간**  .초/초 (.%)  초  -% 

### 정성적 개선

**시청자 경험**
-  "같은 말 왜 두 번 나오지" (혼란 점)
-  "깔끔하고 읽기 편하네" (만족 점)

**완주율 기여**
- 자막 중복 → 이탈률 +% (현재)
- 자막 깔끔 → 이탈률 -% (개선)
- ** 효과 +% 개선**

**도파민 효과**
-  효과 단독 표시 → 강조 효과 +%
- 일반 자막 가독성 +% (분할 개→개)
