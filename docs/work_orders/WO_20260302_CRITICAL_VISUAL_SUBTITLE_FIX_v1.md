##  문제 요약 (사용자 피드백)

### . 자막  치명적 문제 
 "자막 글씨 너무 커, 위아래 좌우 다 짤려서 자막으로 잘 안 보여"
 "자막이 보여지는 여백 좌우로 더 넓혀"

**영향도**  세대 가독성 % → 영상 품질 심각한 타격

### .  도파민 비주얼 없음 
 "앞에 도입부분에 무조건 주제에 대한 도파민이 나오는 여행지가 계속 나와야하는데"
 "이탈율 이거는 심해질수밖에 없는데"

**영향도**  초 이탈율 % 예상 (치명적)

**문제 원인**
-  화면에서  폰트 +  폭 → 좌우  여백 (부족)
-    기준 설계였으나, 실제로는 여백 더 필요
-  세대는 큰 글씨 선호하지만, 짤리면 오히려 가독성 하락

**예상 개선 효과**
- 폰트 크기 → (-%) 가독성 유지하면서 짤림 해소
- 폭 → (+%) 좌우 여백  →  (충분)

##  위험 요소 및 대응

### 리스크  등급 하락
- **위험도** 낮음 (%)
- **원인** 구조 변경 없음, 설정값만 조정
- **대응** 렌더 후 등급 재검증, 점 미달 시 롤백

### 리스크  렌더 시간 초과
- **위험도** 중간 (%)
- **원인** 자막 폭 증가로 렌더 부하 +%
- **대응** .초 → .초 예상, 초 초과 시 폰트 로 재조정

### 리스크   이미지 없음
- **위험도** 낮음 (%)
- **원인** 산토리니/베니스 이미지 부족
- **대응**  전용 폴더에 이미지 + 확보 완료
