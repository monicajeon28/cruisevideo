##   

### 목표
- **완주율** % → % (+%, +%)
- **전환율** .% → .% (+.%, +%)
- **등급 달성률** % → % (+%)
- **도파민 점수** / → / (+점)

### 핵심 전략
개 전문 에이전트가 **병렬로** 각자의 영역을 최적화하고, 결과를 통합하여 도파민 % 영상 생성

### 성공 기준
-   테스트 영상 편 모두 등급 달성
-   도파민 곡선 평균 % 이상
-   완주율 시뮬레이션 % 이상
-   기항지 비주얼 초 이상
