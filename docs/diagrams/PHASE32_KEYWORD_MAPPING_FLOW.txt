## Example: "산토리니 석양" Search Flow

### Step-by-Step Breakdown

```
1. Keyword Extraction
   ↓
   Input: "산토리니 석양을 감상하고"
   ↓
   intelligent_keyword_extractor.extract()
   ↓
   ├─ _extract_proper_nouns()
   │   └─ "산토리니" ∈ PROPER_NOUNS_PORTS ✓
   │   └─ KeywordResult("산토리니", score=85, category="proper_noun")
   │
   └─ _extract_common_nouns_visual()
       └─ "석양" ∈ COMMON_NOUNS_VISUAL ✓
       └─ KeywordResult("석양", score=75, category="visual_common",
                        emotion="romance", eng=["sunset"])

2. Keyword Expansion
   ↓
   expand_keyword_search("산토리니", emotion=None)
   ↓
   ├─ Original: "산토리니"
   ├─ PORT_MAP: ["santorini", "그리스", "에게해", "블루돔", "석양", "이아마을"]
   └─ Result: ["산토리니", "santorini", "그리스", "에게해", "블루돔", "석양"]

3. Emotion-Based Folder Priority
   ↓
   search_with_emotion_priority("산토리니", emotion="wonder")
   ↓
   EMOTION_FOLDER_PRIORITY["wonder"] = [
     "Hook/",                      ← Search here first
     "알래스카 스캐그웨이/",
     "그리스 산토리니/",            ← Or here (port match!)
     "Footage/",
     "크루즈정보사진정리/"
   ]

4. Folder Search
   ↓
   FOR folder IN priority_folders:
     search_in_folder(folder, "산토리니")
     ↓
     ├─ Folder 1: "Hook/"
     │   └─ Files: [sunset_001.mp4, ocean_drone.mp4, ...]
     │   └─ Match "santorini"? → NOT FOUND
     │
     ├─ Folder 2: "알래스카 스캐그웨이/"
     │   └─ Files: [glacier_001.jpg, whale_watching.mp4, ...]
     │   └─ Match "santorini"? → NOT FOUND
     │
     └─ Folder 3: "그리스 산토리니/"
         └─ Files: [bluedome_sunset_001.jpg, oia_village.jpg, ...]
         └─ Match "santorini"?
             ├─ "bluedome_sunset_001.jpg"
             │   └─ difflib("santorini", "bluedome_sunset") = 0.35 ✗
             ├─ "oia_village.jpg"
             │   └─ difflib("santorini", "oia_village") = 0.28 ✗
             └─ Folder name: "그리스 산토리니/"
                 └─ "산토리니" in folder_name ✓
                 └─ FOUND: "그리스 산토리니/bluedome_sunset_001.jpg" ✓

5. Result
   ↓
   asset = "D:/AntiGravity/Assets/Image/크루즈정보사진정리/그리스 산토리니/bluedome_sunset_001.jpg"
   tier = 1 (Local)
   emotion_match = True (wonder ← wonder)
   cost = $0.00
```
