##  핵심 원리

### 도파민 트리거 요소

. **구체적 여행지**  추상적 키워드
   -  "산토리니" (점)   "" (점)

. **시각적 랜드마크**  일반 풍경
   -  하얀 집 + 블루돔 + 석양 (점)   일반 크루즈선 (점)

. **감정 조합** (경이감 + 낭만 + 럭셔리)
   -  산토리니 인피니티풀 (점)  경이감(항공샷) + 낭만(석양) + 럭셔리(성급)

### 세대 특화

- **편안한 접근성** 계단 적음, 평지 (베니스 곤돌라)
- **문화유산** 역사 교육, 자녀 자랑 (산마르코 광장)
- **럭셔리 정당화** "인생 여행" 프레임 (인피니티풀)
- **안전감** 유명 관광지,  (두브로브니크 성벽)
