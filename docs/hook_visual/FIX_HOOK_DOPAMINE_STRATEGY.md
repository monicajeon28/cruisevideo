## 근본 원인 분석

### 문제  키워드 우선순위 오류 

** 키워드 배열 순서**

"" 
  "",      //  순위 긴급성 (추상적, 비주얼 없음)
  "",        //  순위 돈 (추상적, 비주얼 없음)
  "산토리니",     //  순위 핵심 도파민 ( 순위 낮음)
  "",
  "베니스",
  "",
  ""



** 동작 방식**
- 키워드 배열 앞에서부터 순회
- "", "" 먼저 매칭 시도 → **실패** (에셋 없음)
- 매칭 점수 점으로  루프 진입

### 문제   단계 의 역설 

**현재 로직** (  -)


  기항지명 추출 →  이미지 검색
  ↓ 실패 / 키워드가  매칭 방해
     영상 검색
  ↓ 조건   "산토리니", ... →  추출 실패로 스킵
   폴더 의미적 매칭
  ↓ 실패 / 키워드 우선 매칭
   폴더 가중치 랜덤 ( % 우선)
  ↓ 성공  기항지 무관한 "골든파티", "샴페인바" 선택
  전체  검색
  ↓ 도달 안 함


**결과** 에서 **골든파티/샴페인바/댄스파티** 영상이 나옴 → 산토리니 초 노출

### 문제   추출 로직 취약점

**() 동작**
.  전체 에서  (개) 검색
.  섹션 도 검색하지만 **우선순위 낮음**

**  .의 경우**
-   "선착순 명, 만원 지원. 산토리니, 베니스. 월 만원"
- 산토리니/베니스 **존재**하지만,  배열에서 -번째 → 추출 실패

## 도파민 트리거 이미지 우선순위

###   (-점) 즉시 매핑 필수

 기항지  핵심 비주얼  키워드  우선순위 
---------------------------------------
 **산토리니**  하얀 집 + 블루돔 + 석양  오이아, 이아, 칼데라, 에게해, 클리프   
 **베니스**  곤돌라 + 대운하 + 산마르코  운하, 리알토, 수상도시, 광장   
 **두브로브니크**  성벽 + 구시가지 항공샷  스트라둔, 아드리아해, 킹스랜딩, 중세   

###   (-점) 주요 판매 상품

 기항지  핵심 비주얼  키워드 
-----------------------------
 **코토르**  피요르드 + 성벽  몬테네그로, 베이, 요새 
 **스플릿**  디오클레티안 궁전 + 해변  크로아티아, 구시가지, 리바 
 **나가사키**  평화공원 + 야경  하우스텐보스, 사세보, 그라바엔 

###   (-점) 보조 상품

- 후쿠오카, 오키나와, 상하이, 홍콩, 알래스카 주노, 시애틀
