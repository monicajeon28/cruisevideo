## 최종 검증 체크리스트

###   (필수)
-    순서 변경 완료
-    -초 산토리니/베니스 비주얼 노출
-   골든파티/샴페인바 미노출

###   (필수)
-   () 구현
-   회 테스트 중 회 이상 기항지 추출 성공
-   로그 --- 출력 확인

###   (선택)
-    개 기항지 매핑
-    개 등록 ( )
-    도파민 점수 점 이상 (수동 평가)

---

## 긴급 연락처

**문제 발생 시**
.   실패 →  로 즉시 전환 (키워드 재정렬 효과 미미)
.   실패 → 로그 분석 ( 확인)
.   에셋 부족 →    다운로드 ()

**성공 기준**
-  + 완료 시 이탈률 % 이하 달성
-   완료 시 이탈률 % 이하 달성 + 월 수익 만원
