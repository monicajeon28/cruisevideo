#  - 자동 태깅 시스템

   기반 이미지/영상 자동 메타데이터 생성 시스템

## 개요

는 ,개의 크루즈 에셋(이미지, 영상)에 자동으로 태그를 달아 검색 가능하게 만드는 시스템입니다.

### 주요 기능

- **자동 키워드 추출** 이미지/영상 분석 후 -개의 관련 키워드 자동 생성
- **스마트 분류** 카테고리(, ,  등) 자동 분류
- **페르소나 매칭** 타겟 고객층( 시니어, 대 가족, 대 여행자) 자동 매칭
- **영상 프레임 분석** 키프레임 추출 후 대표 장면 분석
- **배치 처리** 대량 파일 일괄 자동 태깅
- **스마트 캐싱** 중복 처리 방지 및 속도 향상
