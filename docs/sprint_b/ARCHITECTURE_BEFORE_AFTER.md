## 핵심 차이점 요약표

 항목      개선율 
-----------------------------
 **생성 방식**  학습 데이터만  학습 데이터 + 개 검증 모듈  +% 정확도 
 ** 생성**  없음   (//초)  +∞% 
 **- 생성**  타이밍 부정확   (/초)  +∞% 
 ** 구조**  단순 텍스트   단계  +∞% 
 ** 주입**  없음   자동 주입  +∞% 
 **키워드 밀도**  낮음   강화  +% 
 **금지어 필터**  없음   자동 교체  +∞% 
 **평균 점수**  ./  ./  +% 
 **등급 달성률**  %  %  +∞% 
 **재시도 횟수**  회 (분)  회 (분)  -% 시간 
 **월 수익**  원  만원  +∞%
