## . 문제 근본 원인 분석

### 현재 상태 (  .)
- **등급 점수** ./점 (개 기항지)
- **실제 기항지 비주얼 노출** .초 (추정)
- **목표 기항지 비주얼** 초
- **** -.초 (-%)

### 로그 분석 결과 (.)

**기항지 키워드 감지**
- 산토리니, 베니스 개 감지 
- 에서 기항지명 추출 성공 
- 이미지 매칭 성공 ( 개 작동 중) 

**기항지 이미지/비디오 실제 사용**

 (초)     산토리니.        .초
- (.초) 그리스 산토리니.                          .초
- (.초) 산토리니.        .초
- (.초) 산토리니.                                .초
- (.초)  크루즈닷 후기. (산토리니 키워드 있음)     후기 이미지 (기항지 )
 (.초)   골든파티  벨리시마.                   크루즈 선박 (기항지 )

총 기항지 비주얼 . + . + . + .  .초 ( 초 미달)


### 근본 원인 가지

####  ** 로직 결함** ( , % 기여)
**파일** .  -

**문제점**

#  필터링 / 세그먼트만 카운트 (// 제외)
  .("", "")
   "", "", "", ""
      #   초를 카운트 안 함!


**실제 영향**
-  초 (산토리니 비디오 .초) → **카운트 제외** 
-  세그먼트만 카운트 → .초만 인식
- /도 기항지  있는데 제외 

**증거**

// .  -

  "" "",
  "" "산토리니", "", "베니스", "", "",
  "" .
  //  가 이 초를 무시함!



####  ** 설계 미스매치** ( , % 기여)
**파일** .

**문제점**
- ** 예상** .초 (산토리니 회)
- **실제 측정** .초 (회만 카운트)
- **** -.초 (-%)

**카운트 불일치 이유**

// 가 제외한 세그먼트들

  "" "",           //  .초 제외
  "" "산토리니", "베니스"
,

  "" "",            //  .초 제외
  "" "산토리니", "", ""
,

  "" "",          //  .초 제외
  "" "산토리니", "", ""


// 실제 기항지 세그먼트 개 (.초)
//  카운트 개 (.초) → -초 손실!


####  **/ 후기 이미지 우선** ( , % 기여)
**파일** .  - (-)

**문제점**

# -   후기 이미지 우선 사용
   '', '', ''
      .(...)
     
           #  기항지 이미지보다 우선!


**실제 영향**

- (.초) '산토리니', '', ''
              → 후기 이미지 우선 선택 (크루즈닷 후기.)
              → 기항지 비주얼 -.초 손실 

 (.초) '산토리니', '', ''
              → 후기 우선 또는 크루즈 선박 비디오
              → 기항지 비주얼 -.초 손실
