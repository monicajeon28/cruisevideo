### .   신규 작업 항목 (품질 강화 단계)

 /가 **등급 달성**(점+)에 집중한다면,
 는 **완주율/전환율 극대화**(품질 강화)에 집중해야 합니다.

####   재정의 목표
- 등급 점 → **점 만점** (최상위 품질)
- 완주율 % → **%** (+%)
- 도파민 리듬 강화 (시각 다양성 + 자막 정확도)
