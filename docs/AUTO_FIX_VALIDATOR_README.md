# -  구현 완료

## 목적
나쁜 콘텐츠를 버리지 말고 즉시 자동 수정하여 **재생성 횟수 % 감소**

## 구현 완료 사항

### . 핵심 파일

#### //.
- ** 클래스** / 자동 수정 엔진
- **()** 을 /점 이상으로 자동 수정
- **()** 를 /점으로 자동 수정
- **()**  전체 자동 수정

#### . ( -)
-   직전에 -  통합
- / 자동 수정 후 재검증
- 승인되지 않은 경우에만 작동

### .  자동 수정 로직


 (,  ,    )
    """
    가지 수정 패턴 적용
    . 금지된 시작 단어 제거 (안녕하세요/여러분/오늘은)
    .   주입 (모르면/놓치면/후회)
    . 구체적 숫자 주입 (만원/가지)
    . 반전 요소 주입 (비밀/사실은/실제로)
    .   추가 (왜/무엇/어떻게)
    """


**수정 예시**
- 원본 "안녕하세요 여러분, 오늘은 크루즈 여행 팁을 알려드릴게요"
- 수정 "왜 이거 모르고 크루즈 타면 만원 실수 비밀"
- 점수 / → /

### .  자동 수정 로직


 (,  ,    )
    """
    가지 수정 패턴 적용
    . - 마커 추가
    . "댓글" 키워드 추가 (점)
    . "공유" 키워드 추가 (점)
    . "정보"/"확인" 키워드 추가 (점)
    . 판매 단어 제거 (상담/예약/신청/전화)
    """


**수정 예시**
- 원본 "지금 바로 전화 상담 예약하세요!"
- 수정 "- 더 많은 정보를 댓글로 공유해주세요. 확인해보세요!"
- 점수 / → /

## 문제 해결

### . -가 작동하지 않아요
****   인 경우에만 작동합니다.  를 통과한 경우 수정하지 않습니다.

### . 수정 후에도  를 통과하지 못해요
**** -는 /만 수정합니다. 크루즈 키워드나 실용 정보가 부족한 경우 여전히 실패할 수 있습니다.

### . 너무 많이 수정되어 원본과 달라요
**** 를 낮춰서 최소한의 수정만 적용하세요
