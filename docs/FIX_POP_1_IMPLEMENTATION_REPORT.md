### . () 메서드 교체

**파일** .  

** (기존 방식)**

- 전략적 위치(초, 초, 초)에 가장 가까운 세그먼트 선택
- 숫자 패턴 정규식 기반  텍스트 추출

** (-- 통합)**

# -- -- % 후킹  시스템 통합

      ()

    #  시퀀스 생성 ( + 보장)
      .(
        (.('', )),
        ''
    )

    # 정밀 타이밍 (.초 이내)
      ., ., .

   
    #    
      


**핵심 변경점**
. 로 개  자동 생성

. 후킹 기법 자동 적용
   -   (초) 희소성 "선착순 명"
   -   (초) 손실 회피 "만년 빙하 녹는다"
   -   (초) 대비 효과 "패키지   "
. 기존 방식  보장 (오류 발생 시)

##  주의사항

### .   폴더 구조 확인 필요

현재 코드에서 가정한 폴더 구조

////
├── 가격비교/
├── 특가/
├── 후기/
├── 가격표/
└── 긁급/


**실제 폴더 구조와 다를 수 있음** → 런타임 테스트 필요

**해결책**
. 실제 폴더 구조 확인 후  매핑 수정
. 폴더 없을 시 차  (일반 이미지 검색) 동작

### .  오류 시  보장


   
    .("--  실패, 기존 방식 사용 ")
      


-  오류 발생 시 기존 방식 자동 전환
- 파이프라인 중단 없이 계속 동작

### .  과의 중복 방지

현재 구현
-   (.초, .초, .초) - 유지
- 전략  (.초, .초, .초) - -- 적용

**겹침 없음 확인**  최소 .초 간격 유지

##  에러 처리

### .   실패
- **원인** 모듈 경로 오류, 의존성 누락
- **처리** 기존 방식 
- **로그**   실패, 기존 방식 사용

### .  폴더 없음
- **원인** 실제 폴더 구조와 불일치
- **처리** 차  검색 → 차 일반 검색
- **로그**    → 이미지명

### .  이미지 매칭 실패
- **원인** 키워드 매칭 점수 부족, 이미지 부족
- **처리**  텍스트만 표시 (이미지 없이)
- **로그**   매칭 실패 ,
