**파일** .

-    - (- 간격 검증 추가)

    ( - )  ,   (, )
   (      )
       - 
      "".("- 최소 간격 초 미만 (충돌 위험)")

  # 기존  , 
  # 변경  , ,   #  -- 충돌 검증용
  
-     호출부 수정
  
  # 기존 ,   .(, )
  # 변경
  , ,   .(, )
  
-   저장

    ./ (개)  #  만점
