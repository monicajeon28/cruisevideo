## - 타이프라이터 글자 간격 수정 ()

### 문제 원인

# 현재  * .   * .   (글자 겹침!)
  (( / ) + ( -  / ) * (. * .))

# 한글은 정사각형에 가까움 → .이 아니라 . 필요


### 수정 파일
.

### 수정 내용

**위치** -행


#  수정 전 
  ()
  (( / ) + ( -  / ) * (. * .))

#  수정 후 
  ()
# 한글은 정사각형에 가까움 (폭/높이 비율 ≈ .)
  .  # 한글 글자 폭 비율
  . * 
  (( / ) + ( -  / ) * )


### 검증 방법
- 생성된  파일에서 (,) 값 확인
- 글자 간격이 약  ( * .)가 되어야 함

## 작업 순서 권장


일차 ( - )
├── -  →  역갱신 (분)
├── - 타이프라이터 글자 간격 (분)
└── 테스트 및 검증 (분)

일차 ( - )
├── - 출력 폴더 구조 (분)
├── - 썸네일 헤드카피 (분)
└── 테스트 및 검증 (분)

일차 ( - )
├── -  메시지 증가 (분)
├── - 누끼 이미지 선택 (분)
└── 전체 통합 테스트 (시간)

## 테스트 체크리스트

###  검증
-   자막이 나레이션과 동시에 시작/종료되는지
-   타이프라이터 효과에서 글자가 겹치지 않는지
-   줄 자막이 올바르게 표시되는지

###  검증
-   별 폴더가 생성되는지
-   모든 관련 파일이 같은 폴더에 저장되는지
-   썸네일 헤드카피가 후킹 문구인지 ( 그대로 아님)

###  검증
-   초 영상에 이 개 이상인지
-   - 포인트(초, 초, 초)에 이 있는지
-   썸네일 누끼 이미지가 콘텐츠와 관련 있는지
