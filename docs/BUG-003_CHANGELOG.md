## 성능 개선

###  (정규식 모드)


입력 "크루즈 뷔페에서 맛있는 음식을 먹었어요"
출력 "뷔페에서", "맛있는", "음식을", "먹었어요", "크루즈"


- 조사 포함 "뷔페에서", "음식을"
- 어미 포함 "맛있는", "먹었어요"
- 정확도 %

###  (형태소 분석 모드)


입력 "크루즈 뷔페에서 맛있는 음식을 먹었어요"
출력 "크루즈", "뷔페", "음식"


- 조사 제거 
- 어미 제거 
- 정확도 %
