##  발견된 이슈 (건)

### . - 미적용  
- **문제**  초기화 성공했으나 실제 - 개 생성
- **원인** 스크립트에 - 세그먼트 없음 (자동 생성 로직 미작동)
- **영향** - 점수 / (-점)
- **해결** .에서  호출 로직 확인 필요

### . 스크립트 길이 초과  
- **문제**  계열 스크립트 초 (초 초과)
- **원인**  단계 추가로 스크립트 길이 증가
- **해결** . 사용 (.초)

### .  감소  
- **문제** . → . (-.점)
- **원인** 스크립트 변경 ( / )
- **영향** 경미 (전체 점수 +. 유지)
