# -- 테스트 계획

## 목표
-자막 싱크 % 달성 검증

## 테스트 시나리오

###     . 전체 렌더링
- 파일 //.
- 예상 결과
  - 자막 정확도 % (오차 .초 이내)
  - 완주율 +% (% → %)
  - 자막 늦게 뜨기/일찍 사라지기 건

###     영상 회
- 명령어  . --  -- 
- 검증 항목
  .   수집 정상 동작
  .  모듈  성공
  . 자막 재조정 로그 출력
  . 검증 결과 

###     (기항지 개)
- 명령어  . --  -- 나가사키 -- " 벨리시마" -- 기항지정보
- 검증 항목
  . 단일 기항지 영상 생성
  . 자막 싱크 % 달성

## 성공 기준

### 정량 지표
-   자막 정확도 % (''  )
-   평균 오차 ≤ .초 ('' ≤ .)
-   최대 오차 ≤ .초 ('' ≤ .)

### 정성 지표
-   자막이 와 동시에 표시됨 (.초)
-   자막이  끝나기 전에 사라지지 않음
-    구간 자막도 정확히 싱크됨

## 예상 소요 시간
-   분 (렌더링 분 + 검증 분)
-   분 (렌더링 분 + 검증 분)
-   분 (렌더링 분 + 검증 분)
- **총합 분**

## 문제 발생 시 조치

###  발생

    ''

→ /. 파일 존재 여부 확인

### ''  
- 로그에서  확인
- 해당 세그먼트의    자막  비교
-  매핑 오류 확인

### 렌더링 크래시
- 로그에서 마지막  확인
-  에러 메시지 확인
- 임시 파일 삭제 후 재시도
