#   - 자동 삽입 - 빠른 참조 가이드

## 

**문제** 가 -을 % 확률로 생략 → - ./ 실패
**해결** () 자동 삽입 로직 구현 → - ./ 보장
**결과** 등급 달성률 % → %+, 월 만원 절감

---

## 핵심 구현 (분 요약)

### 신규 메서드 개 (.)


# . () - 메인 로직 ( -)
 (,  ,  ) - 
    """- 개 보장 (개 → 개 자동 삽입)"""

    # - 개수 확인
      (      ""  "")

       
           # 통과

    # 부족하면 자동 삽입
    # - 초 - 손실 회피 프레임
    # - 초 - 사회적 증거 프레임

# . () - 세그먼트 생성 ( -)
 (,  ,  ) - 
    """- 세그먼트 생성"""

       ""  # 초
          
            "잠깐! 이걸 모르면 만원 손해보십니다.",
            "여기서 놓치면 만원 날립니다.",
            # ...
        
      # "", 초
          
            "지난달 %가 만족했습니다. 당신만 놓치시겠어요",
            "이미 명이 신청했습니다.",
            # ...
        

     "" "", "" ., "" .()

# . () - 삽입 위치 ( -)
# . () - 타이밍 조정 ( -)
