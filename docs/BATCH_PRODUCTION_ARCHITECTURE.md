### - 스크립트 중복 방지

**상태** 승인됨

**컨텍스트**
개 영상이 모두 다르게 보여야 함

**결정**
단계 중복 체크 정확 매칭 → 조합 매칭 → 의미적 유사도

**결과**
- 장점 ,개 조합에서 개 고유 선택 보장
- 단점 의미 유사도 체크에 시간 소요 (약 .초/스크립트)

---

### -  렌더링

**상태** 승인됨

**컨텍스트**
 렌더링 초/영상 → 개  .시간

**결정**
  인코딩 사용,  프리셋 (품질-속도 균형)

**결과**
- 장점 렌더링 % 단축 (초 → 초)
- 단점   필수

---

### - 개 동시 렌더링

**상태** 승인됨

**컨텍스트**
 ,   환경에서 최적 동시 처리 수

**결정**
렌더링 개 동시,  개 동시, 스크립트 생성 개 동시

**결과**
- 장점 시스템 안정성 유지,  방지
- 단점 이론적 최대 속도보다 느림
