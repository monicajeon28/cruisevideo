### 목표 % (/ 파일 활용)

  - % ( 개 요소 전부)
  - % (단계 전부)
   - % (가지 유형 전부)
   - % (개 패턴 활용)
   - % (하이브리드 전략 + 금지어)
  ,, - % (- + -)
  , - % (  ++)
   - % ( 단계 증폭)
   - % (/ 정보 공개)
   - % (- 감정 곡선)
