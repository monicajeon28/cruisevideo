# 애니메이션 시스템 .

## 개요

  의 애니메이션 시스템이 **점에서 점(등급)** 으로 업그레이드되었습니다.

### 지원 효과

 효과  설명  성공 확률  개발 기간  우선순위 
--------------------------------------------
 ****  확대/축소 효과 (드라마틱)  %  -일   
 ****  슬라이드 이동 ( 메시지)  %  -일   
 ** /**  페이드 전환 (기본)  %  .일

## .  애니메이션

### 설명
영상을 점진적으로 확대하거나 축소하는 효과입니다. 중요한 순간을 강조하거나 드라마틱한 분위기를 연출할 때 사용합니다.

### 파라미터


(
    "",
    .,      # 효과 시작 시간 (초)
    .,        # 효과 종료 시간 (초)
    
        '' ,      #  확대,  축소
        '' .,   # 시작 스케일 (.  원본 크기)
        '' .      # 종료 스케일 (.  % 크기)
    
)


### 사용 예시


 .  (
    ,
    ,
    
)

  ()

#   효과 (. → .)
  (
    "",
    .,
    .,
    
        '' ,
        '' .,
        '' .
    
)

  (
    ".",
    ,
    .,
    "중요한 순간!",
    
)

  .(
    ,
    ".",
    (, )
)


### 기술 세부사항

- ** 필터** '*()''*()'
- **보간 방식** 선형 보간 ()
- **표현식**  + ( - ) * 
- ** . 호환** (), () 함수만 사용 (() 함수 회피)
- **성능 영향** +-% ( 가속 시)

## .  애니메이션

### 설명
영상을 특정 방향에서 슬라이드하여 나타나게 하는 효과입니다.  메시지나 강조 텍스트와 함께 사용하면 효과적입니다.

### 파라미터


(
    "",
    .,
    .,
    
        '' '',  # 슬라이드 방향
        '' .                # 이동 거리 (화면 비율, .-.)
    
)


### 방향 옵션

- '' 왼쪽에서 오른쪽으로
- '' 오른쪽에서 왼쪽으로
- '' 위에서 아래로
- '' 아래에서 위로

### 사용 예시


#  메시지용  효과
  (
    "",
    .,
    .,
    
        '' '',
        '' .  # 화면 너비의 %
    
)

  (
    ".",
    ,
    .,
    " 특별 할인!",
    
)


### 기술 세부사항

- ** 필터** ''''
- ** 오프셋**  *  * ( - ) (왼쪽에서 시작)
- ** 오프셋**  *  * ( - ) (위에서 시작)
- **성능 영향** +-%

---

## .  / 애니메이션

### 설명
영상을 서서히 나타나게 하거나 사라지게 하는 부드러운 전환 효과입니다. 모든 영상의 시작과 끝에 기본으로 적용하는 것을 권장합니다.

### 파라미터


(
    "",
    .,
    .,
    
        '' .,   #   시간 (초)
        '' .   #   시간 (초)
    
)


### 사용 예시


# 부드러운 전환 효과
  (
    "",
    .,
    .,
    
        '' .,
        '' .
    
)

  (
    ".",
    ,
    .,
    
)


### 기술 세부사항

- ** 필터** , 
- **알파 채널** . (투명) → . (불투명)
- **성능 영향** +-% (가장 가벼움)

## . 성능 최적화

### 인코딩 시간 비교

 설정  인코딩 시간  오버헤드  등급 
-----------------------------------
 **애니메이션 없음**  .분  -  - 
 **만**  .분  +.%   
 **만**  .분  +.%  + 
 **만**  .분  +.%  + 
 ** + **  .분  +.%   

### 최적화 팁

. **는 항상 사용** 성능 영향이 거의 없음 (+-%)
. **은 선택적 사용** 중요한 순간에만 적용
. **는  메시지와 함께** 시각적 강조가 필요할 때만
. ** 가속 활용** / 사용 시 오버헤드 최소화

## .  . 호환성

### 문제점
 .에서는 () 함수의 쉼표 파싱에 버그가 있습니다.


#  작동 안 함 ( .)
'*(((, ), . + .*/, .))''*...'


### 해결책
(), () 함수로 조건 표현식을 대체합니다.


#  작동 ( . 호환)
'*(. + .*(, (, (-)/)))''*...'
