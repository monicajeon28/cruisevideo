#  - - 구현 완료

## 프로젝트 개요

  비동기 호출 최적화를 통해 ** 대기 시간 % 이상 단축**

### 성공 기준 달성

 기준  목표  달성  상태 
------------------------
 병렬 호출  개 동시 처리   로 제한   
     / 준수     구현   
 응답 시간 단축  % 이상   병렬 처리로 %+ 단축   
 캐시 히트율  % 이상    +  캐싱   
 테스트 통과  모든 테스트   개 테스트 작성
