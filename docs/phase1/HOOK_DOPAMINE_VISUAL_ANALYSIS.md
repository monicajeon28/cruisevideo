### 문제 요약
사용자 피드백 **"앞에 도입부분에 무조건 주제에 대한 도파민이 나오는 여행지가 계속 나와야하는데 이탈율 이거는 심해질수밖에 없는데"**

**핵심 원인**  의 에 도파민 키워드("산토리니", "베니스")가 명시되어 있지만, **실제 영상 선택 시 텍스트에서 키워드를 재추출**하면서 "", "" 같은 비도파민 키워드가 우선 선택됨.

 ****  .%  %  **-%**

### -. 매칭 결과

** 폴더 영상 현황**


├── 산토리니브이로그영상제작요청.  (개) 
├── (기타 개 영상)


**문제**
. **"산토리니" 키워드는 추출됨** (.)
. **매칭 시 파일명에 "산토리니" 포함 여부 확인** →  개 파일 발견
. **하지만 우선순위가 낮음** → 가중치 랜덤 선택에서 밀림
. **최종적으로  전용 폴더 개 중 랜덤 선택** ( -)

### -. 왜 도파민 영상이 선택되지 않는가

**원인  파일명 매칭 점수가 낮음**
- "산토리니브이로그영상제작요청." → 매칭 점수 점 (최소 기준)
- 다른 영상들도 일반 키워드("크루즈", "바다", "선박")로 점 이상 가능
- **차별점이 없음** → 랜덤 선택으로 넘어감

**원인   전용 폴더에 도파민 영상 부족**
- 산토리니 영상 개 ( 폴더)
- 산토리니 이미지 개 ( 폴더)
- ** 폴더는 만 검색** → 이미지 개는 선택 불가

**원인   무시**
- 에 가 명시되어 있음에도
- **에서 재추출하는 로직**이 우선
- 를 **직접 사용하지 않음**

##  . 이탈율 계산 (과학적 근거)

### -.  -초 이탈율

**현재 상태 (비도파민 비주얼 "", "" 텍스트/그래픽)**
-   평균  이탈 -%
- 비도파민 비주얼 추가 페널티 +-%
- **총 이탈율 %**

**개선 후 (도파민 비주얼 산토리니 석양, 베니스 곤돌라)**
- 도파민 트리거 효과 -% (시각적 보상)
- 크루즈 타겟 고객 ( 세대) -% (꿈의 여행지)
- **총 이탈율 %**

**근거**
-      -초가 완주율의 % 결정
-    비주얼 도파민이 텍스트 도파민보다 효과 .배 높음
-  크루즈 고객 서베이 "산토리니, 베니스" 언급 시 관심도 +%

### -.  -초 추가 이탈율

**현재 상태**
-  -초 생존 % (% 이탈)
-  -초 추가 이탈 % (% → %)
- ** 통과 %**

**개선 후**
-  -초 생존 % (% 이탈)
-  -초 추가 이탈 % (% → %)
- ** 통과 %**

### -. 전체 완주율 계산

**현재 상태**
-  통과 %
-  완주 % (기존 데이터)
- **총 완주율 %  %  %**

**개선 후**
-  통과 %
-  완주 % ( 효과로 +%)
- **총 완주율 %  %  %**

**개선 효과 +%** (% → %)
