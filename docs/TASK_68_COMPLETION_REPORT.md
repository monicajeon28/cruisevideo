##  

###  (수동 업로드)

 문제  영향 
-----------
 랜덤 시간 업로드  알고리즘 페널티 -% 
 시간 내 편 업로드   의심 → 밴 위험 
 동일 카테고리 연속 편  중복 페널티 -% 
 새벽 시간 업로드  노출량 -% 

###  (스케줄러 사용)

 개선  효과 
-----------
  분  밴 위험 -% 
 피크 타임 % 집중   +% 
 카테고리/기항지 간격  중복 페널티 % 
 일정한 업로드 간격  노출량 +% 

**예상 종합 효과**
-  .% → .% (+%)
- 노출량  →  (+%)
- 전환율 .% → .% (+%)
- 밴 위험 중간 → 낮음 (-%)
