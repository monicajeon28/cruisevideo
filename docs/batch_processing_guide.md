# 배치 처리 전략 가이드

## 개요

등급 영상 개를 안정적으로 생산하기 위한 개배치 처리 전략입니다.

### 주요 특징

- **배치 분할** 개를 개씩 번으로 나누어 처리
- **메모리 관리** 배치마다 .() 및 임시 파일 정리
- **체크포인트** 중간 저장으로 실패 시 재시작 가능
- **진행률 표시** 실시간 진행률 및 예상 완료 시간

---

## 배치 처리 흐름


  콘텐츠 선택
  └─ 에서 개 아이디어 선택
  └─ 개씩 개 배치로 분할

 - 배치별 처리
  ┌─ 배치 / (개)
  │   ├─ 병렬 생산 ( )
  │   ├─ 품질 검증 (등급 필터링)
  │   ├─ 체크포인트 저장
  │   ├─ 메모리 정리
  │   └─ 진행률 표시
  │
  ├─ 배치 / (개)
  │   └─ ... (반복)
  │
  └─ 배치 / (개)
      └─ 최종 체크포인트

  최종 정리
  ├─ 통계 계산
  ├─ 리포트 저장
  └─ 체크포인트 정리

## 기존 함수와 비교

 기능  ()  () 
--------------------------------------------------------------------
 배치 분할   없음   개씩 분할 
 체크포인트   없음   배치마다 저장 
 메모리 정리   없음   배치마다 정리 
 재시작   처음부터   마지막 배치부터 
 진행률 표시   개마다   배치마다 상세 
 예상 완료 시간   없음   실시간 계산

## 결론

배치 처리 전략을 사용하면

. **안정성 향상** 메모리 관리로 크래시 방지
. **재시작 가능** 체크포인트로 중단된 작업 재개
. **투명성** 상세한 진행률 및 예상 시간 제공
. **유연성** 다양한 환경에 맞게 조정 가능

**권장 사용법** 개 이상 대량 생산 시 배치 처리 전략 사용
