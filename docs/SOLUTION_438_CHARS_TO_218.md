# 자 → 자 해결 방안 (만 뷰 최적화)

## 문제 진단

### 현재 상태
- 가 자(초) 생성 → 초 목표 배 초과
- 에 자 제한 명시했지만 **무시됨**
- 등급 ./ 달성했지만 **렌더링 실패**

### 근본 원인
**는 자연어 프롬프트의 "숫자 제약"을 확률적으로 해석**
- "자 이내"를 "대략 자" 정도로 이해
-   같은 구조적 제약 없음
- 긴 프롬프트(,자+)에서 숫자 제약은 우선순위 낮음

## 만 뷰 체크리스트 (최종)

### 콘텐츠 품질
-    -초 만원  비주얼 (- )
-    -초 손실 회피 ("날립니다")
-    -초 대안 제시 (월 만원 산토리니)
-   - 개 초, 초, 초
-    개 초, 초, 초, 초
-    단계 긴급성 + 행동 + 신뢰
-    개 이상 년 + 억 보험

### 기술 품질
-   총 글자 수 ≤ 자 ( 초 보장)
-   등급 점수 ≥ 점
-   렌더링 시간 ≤ 초 (이미지 자막)
-   영상 길이 초 초

### 업로드 최적화
-   썸네일 만원  빨간색
-   제목 "여행사 만원 손해  크루즈 월 만원"
-   첫 댓글 " 프로필 링크에서 만원 특가"
