# --    읽기

## 문제 상황


 예측 .초 ()
 실제 .초 (실제 파일)
손실 .초 (%)


### 원인
-  캐시 히트 시 () 메서드 사용
- 실제 오디오 파일의 길이를 읽지 않음
- 예측 이 실제와 최대 % 차이 발생
- 영상 타이밍 불일치 문제 유발

## 수정 포인트

### .  실행
- 캐시된 파일의 실제 을 로 읽기
- .초 정밀도로 정확한 값 반환

### .  로직
-  실패 시 기존 () 사용
- 가지 예외 처리
  - . 초 
  -   파싱 실패
  -   미설치

### . 로깅 개선
-  성공 (.초, )
-  실패 (.초,  - )
- 캐시 히트 경로 명확히 구분

### . 신규 생성 경로
- 기존 로직 유지 ( 헤더에서  반환)
- 캐시 히트/미스 두 경로 모두 처리
