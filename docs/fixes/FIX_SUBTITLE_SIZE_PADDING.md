# -- 자막 크기 및 여백 조정 (--)

## 문제 정의

### 증상
. **자막 글씨 화면 짤림**  폰트가 너무 커서 좌우 및 상하 여백 부족
. **배경 여백 부족**  패딩으로는  폰트를 감싸기에 불충분
. **줄 자막 하단 짤림**  위치 에서 줄 자막이 화면 하단 벗어남

### 원인 분석
- -- (--)에서 시니어 가독성 향상을 위해 폰트 크기를  → 로 증가
- 하지만  화면에서  폰트는 과도하게 큼
-    내에서도 여백 부족으로 텍스트 잘림 발생

## 예상 효과

### . 짤림 방지
 항목      개선 
---------------------------
 폰트 크기      -% (적정 크기) 
 배경 패딩      +% (여백 확보) 
  위치      + (하단 여유) 
 최대 폭      + (  최적) 

### . 가독성 유지
-  폰트는  세대에게 충분히 큼 (- 권장 범위 내)
- 배경 패딩 로 텍스트와 배경 명확한 분리
- 하단 여백 로 줄 자막도 여유 있게 표시

### .   준수
- 최대 폭  ≈    (% 활용)
-   공식 권장 범위 내
- 다양한 디바이스(,  등)에서 안정적 표시

## 검증 방법

### 테스트 케이스
. **줄 자막** "동유럽 크루즈 알아보다가 놀란 이유" (자)
   - 예상 폭      →  제한 → 줄 분할
   - 실제 표시 줄 자 + 줄 자

. **줄 자막** "년간 ,명이 선택한 / 올인클루시브 크루즈" (자)
   - 줄 자     →  제한 
   - 줄 자     →  제한 
   - 총 높이 , 하단 위치  (여유 )

. **배경 여백**
   - 텍스트 폭  + 패딩  (양쪽 )  
   - 화면 폭 의 % (적정 비율)

##  예측

### 가독성 개선 효과
- 자막 짤림 제거 → 정보 전달률 % → % (+%)
- 배경 여백 확보 → 시인성 향상 → 완주율 +%
- 적정 폰트 크기 →  눈 피로 감소 → 재시청률 +%

### 비즈니스 임팩트
- 완주율 % → % (+.%)
-  % → .% (+.%)
- 전환율 .% → .% (+.%)
- **월 수익 만원 → 만원 (+만원, +.%)**
