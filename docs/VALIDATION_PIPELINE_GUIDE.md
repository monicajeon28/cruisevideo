# - 자동 검증 파이프라인 가이드

## 개요

- 영상 자동 검증 파이프라인은 영상 생산 즉시 품질을 자동으로 판정하고, 승인/거부/수동검토를 결정하는 시스템입니다.

---

## 아키텍처


영상 생산 완료
       ↓
  로드
       ↓

  - 특징 벡터 추출
  -  모델 예측
  - 기준별 점수 계산
       ↓

  - 임계값 체크
  - 승인/거부/수동검토 결정
       ↓

  -  저장
  - 알림 전송
  - 재생산 큐 이동
       ↓
결과 저장 & 대시보드

## 트러블슈팅

### . 모델 파일을 찾을 수 없음

**문제**

 모델 파일을 찾을 수 없습니다 ///.


**해결**
- -을 먼저 실행하여 모델 훈련
- 또는  모델 사용 (테스트용)

### .  저장 실패

**문제**

  저장 실패   


**해결**
- 는 동시 쓰기 제한 있음
- 로 마이그레이션 권장
- 또는 배치 크기 줄이기

### . 재생산 큐 무한 루프

**문제**
- 같은 영상이 계속 재생산 큐에 추가됨

**해결**
- 최대 재시도 횟수 (회) 체크
-  증가 확인
- 개선 힌트가 제대로 적용되는지 확인
