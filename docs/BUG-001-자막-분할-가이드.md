# - 수정 자막 라인 길이 제한 로직

## 문제 설명

기존 시스템에서는 자막이 자를 초과하여 화면에서 잘리는 문제가 발생했습니다.
이는 가독성을 저하시키고 테스트 실패의 원인이 되었습니다.

## 해결 방법

### . 자막 분할 함수 구현

/.에 두 가지 핵심 함수를 추가했습니다

#### (, )

텍스트를 최대 길이 이하로 자동 분할하는 함수입니다.

**분할 우선순위**
. 구두점 (., ,, !, , , )
. 조사 경계 (을, 를, 이, 가, 은, 는 등)
. 어절 경계 (공백)
. 강제 분할 (마지막 수단)

**사용 예시**

 .  

  "크루즈 뷔페에서 절대 놓치면 안 되는 음식이 있어요!"  # 자
  (, )

()
# '크루즈 뷔페에서 절대 놓치면', '안 되는 음식이 있어요!'


#### (, )

 객체를 여러 개로 분할하며, 타이밍도 자동으로 조정합니다.

**기능**
- 텍스트 자동 분할
- 타이밍 균등 배분
- 도 함께 분할

**사용 예시**

 .  , 

  (
    "크루즈 뷔페에서 절대 놓치면 안 되는 음식이 있어요!",
    .,
    .
)

  (, )

 ,   ()
    ("+ . (.. - ..)")

#  크루즈 뷔페에서 절대 놓치면 (. - .)
#  안 되는 음식이 있어요! (. - .)


### .  자막 생성 시 자동 적용

.() 메서드가 자동으로 자 제한을 적용합니다.

**주요 변경사항**
-  파라미터 추가 (기본값 )
- 자막 생성 전 자동 분할 처리
- 경고 로그 출력
- 최종 검증 단계 추가

**로그 예시**

- 자막 분할 '크루즈 뷔페에서 절대 놓치면 안 되는 음식이 있어요!' (자) - 개 라인
- 총 개 자막을 자동 분할했습니다
- 총 개 자막 라인 생성 (모두 자 이하)


## 테스트

### 테스트 실행


 /.


### 테스트 커버리지

. **기본 자 제한 테스트** - 자 텍스트를 개로 분할
. **여러 번 분할 테스트** - 자 텍스트를 개로 분할
. **구두점 우선 분할 테스트** - 쉼표에서 자연스럽게 분할
. **짧은 텍스트 테스트** - 자 텍스트는 분할하지 않음
. **정확히 자 테스트** - 자는 분할하지 않음
. **조사 경계 테스트** - 조사 뒤에서 분할
. ** 분할 테스트** - 타이밍 자동 조정 검증

## 성능 영향

- **처리 속도** 분할 로직은 () 복잡도로 매우 빠름
- **메모리** 추가 메모리 사용은 거의 없음
- **자막 품질** 자연스러운 분할로 가독성 % 달성

## 호환성

- 기존 코드와 완전 호환
-  타이밍과 완벽 동기화
-  기능도 정상 작동

## 향후 개선 사항

. 더 정교한 한국어 문법 분석 (형태소 분석)
. 사용자 정의 분할 규칙 추가
. 다국어 지원 강화
