**완벽한 자동 검증 파이프라인 가이드**

---

## . 개요

자동 검증 파이프라인은 영상 생산 완료 후 ** 모델 기반 - 검증**을 통해 **자동 승인/수동 검토/재생산** 결정을 수행합니다.

### 단계 승인 시스템

- **  자동 승인** (%+ -, %+ )
- **  수동 검토** (-% -)
- **  재생산 큐** (% -)

### 예상 성능 지표

- **자동 승인율** -%
- **  비율** % ( % 임계값으로 보장)
- **처리 속도** -초/영상 (개 특징 추출 +  예측)
- **프로덕션 효과** 수동 검토 시간 **% 절감**

이 시스템은 ****, ****, ****, **** 개 핵심 컴포넌트로 구성되어 있으며, 완전한 자동화 워크플로우를 제공합니다.

## . 시스템 구성

### . 핵심 컴포넌트

#### 
- **역할**  모델 기반 - 예측
- **모델** / 분류기
- **입력**   데이터

#### 
- **역할** 단계 승인 결정 (자동 승인/수동 검토/재생산)
- **승인 임계값**
  - 자동 승인 -  %,   %, 모든 기준  %
  - 수동 검토 %  -  %
  - 재생산 -  %
- **피드백** 상세 개선 힌트 자동 생성

#### 
- **역할** 전체 파이프라인 관리 ( 로드 → 검증 → 승인 결정 →  저장 → 알림)

#### 
- **역할** 재생산 큐 관리 (우선순위 기반)

- **우선순위** - 확률에 반비례 (.-.  , .-.  , .  )
- **재시도 제한** 최대 회

## . 단계 승인 시스템

###   자동 승인 (%+ -, %+ )

**조건**
- - 확률  %
- 신뢰도  %
- 모든 개 기준 점수  %

**액션**
- 즉시  업로드 큐로 이동
- 에 '',  저장
-  성공 알림 전송 (옵션)

**예상 비율** -%

**예시**

# 자동 승인된 영상

    "" "",
    "" "",
    "" .,
    "" .,
    "" 
        "" .,
        "" .,
        "" .,
        "" .,
        "" .,
        "" .
    ,
    "" 



###   수동 검토 (-% -)

**조건** ()
- %  - 확률  %
- 신뢰도  % (단, -  %)
- -개 기준만 % 미달 (단, -  %)

**액션**
- 수동 검토 큐로 이동
- 에 '',  저장
- 개선 힌트 자동 생성
-  경고 알림 전송 (옵션)

**예상 비율** -%

**예시**

# 수동 검토 필요

    "" "",
    "" "",
    "" .,
    "" .,
    "" 
        " 개선 '' 또는 '' 타입 사용, 텍스트 오버레이 추가, / 강화",
        " 개선 전환 효과 다양화, 텍스트 오버레이 증가, 컬러 그레이딩 강화"
    ,
    "" 



###   재생산 큐 (% -)

**조건**
- - 확률  %

**액션**
- 재생산 큐로 자동 이동
- 에 '' 저장
- 우선순위 계산 (- 확률에 반비례)
- 최대 회 재시도 제한
-  에러 알림 전송 (옵션)

**예상 비율** -%

**우선순위 계산**
- .-.   (일반)
- .-.   (낮음)
- .   (매우 낮음)

**예시**

# 재생산 필요

    "" "",
    "" "",
    "" .,
    "" .,
    "" ,
    "" 
        " 개선 '' 또는 '' 타입 사용, 텍스트 오버레이 추가",
        " 개선 장면 전환 속도 조정 (-초 균형)",
        " 개선  강화, - 타이밍 추가 (초, 초)",

###    비율은 어떻게 보장하나요
****  % 임계값을 사용하여   비율을 **% 미만**으로 유지합니다. 모델의 확률이 . 이상이더라도, 신뢰도가 . 미만이면 수동 검토로 분류됩니다.

###  처리 속도가 느린 경우 어떻게 개선하나요
**** 현재 단일 영상 처리 시간은 **-초**입니다. 배치 처리 시  ()를 사용하면 **배 속도 향상**이 가능합니다. 를 사용하는 경우 더욱 빨라집니다.
