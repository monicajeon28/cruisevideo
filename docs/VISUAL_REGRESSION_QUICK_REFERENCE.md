##  시니어 육안 테스트

### 준비물
- 렌더링된  영상
- 모바일 디바이스 ( ,  )
-  시니어 명 (-세)

### 질문지

. 자막을 쉽게 읽을 수 있나요
     매우 쉬움 ( 폰트 적합)
     보통 (읽을 수 있음)
     어려움 (폰트 작음)

.  메시지가 눈에 띄나요
     명확히 보임 ( 폰트 적합)
     보통
     눈에 안 띔

. 이미지가 내용과 관련 있나요
     매우 관련 있음 (% 이상)
     보통 (-%)
     관련 없음 (% 미만)

. 영상이 선명한가요
     매우 선명 (  수준)
     보통 (  허용)
     흐림 (  부적합)


**합격 기준**
- 질문 - / 명 "매우 쉬움" 또는 "명확히 보임"
- 질문  / 명 "매우 관련 있음"
- 질문  / 명 "보통" 이상
