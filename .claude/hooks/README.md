#     시스템

## 개요

서브에이전트가 **병렬로 실행**될 때 자동으로 적절한 을 트리거하는 지능형 라우팅 시스템입니다.

## 구조


.//
├── .           #  메인 라우터 (에이전트 →  자동 매칭)
├── /                     # 전역  (모든 에이전트 공통)
│   ├── -.            # 실행 전
│   ├── -.           # 실행 후
│   └── -.          # 에러 발생 시
├── /                     # 코딩 카테고리 
│   ├── -.            # 컨텍스트 최적화, 스킬 로드
│   └── -.           # / 자동 체크
├── -/               # 크루즈 도메인 특화 
│   ├── -.            #   자동 로드
│   └── -.           # 도메인 품질 검증
├── /                    # 유튜브 카테고리 
│   ├── -.            #  할당량 체크, 학습 데이터 로드
│   └── -.           # 생산 로그, 학습 사이클 트리거
├── /                   #  카테고리 
│   ├── -.            # 환경 변수 체크, 백업 경고
│   └── -.           #  작업 로그
├── /                     # 리뷰 카테고리 
│   └── -.           # 리뷰 결과 저장
└── /                    # 테스팅 카테고리 
    └── -.           # 커버리지 리포트


##  실행 흐름


 
    사용자 요청 --  
     -- 에이전트 카테고리 판별
     -- 전역 -
     -- 카테고리별 -
     -- 에이전트 실행
     -- 성공
     -- 전역 -
     -- 전역 -
     -- 카테고리별 -
     -- 완료
     --

**-**
- 완료 로그
- 성능 메트릭 저장

- 에러 로그

- - 자동 호출 권장

## 추가된 고급  (권장)

### . -  (선택적)

**목적** 토큰 최적화를 위한 선택적 컨텍스트 로드


# .///-.
# - 기반으로 필요한 파일만 로드


### . -  ( 전용)

**목적** 학습 데이터 피드백 수집


# .///-.
# 성과 데이터를 수집하여 다음 생산에 반영


### . -  (/)

**목적** 보안 취약점 자동 검사


# .///-.
# 하드코딩된 시크릿 탐지
#   패턴 체크


### . -  ()  구현됨

**목적** 코드 품질 게이트

**자동 검증 항목**
- 함수 복잡도 (줄 이하)
- 테스트 커버리지 (% 이상 권장)
-  경고 (개 미만 권장)
- 종합 점수 / 이상 → 

### . -   구현됨

**목적** 토큰 절약을 위한 선택적 컨텍스트 로드

**기능**
- - 기반 선택적 로드
- 에이전트별 맞춤 컨텍스트 추출
- 토큰 % 절감 효과

### . -   구현됨

**목적** 보안 취약점 자동 검사

**검사 항목**
- 하드코딩된 시크릿 탐지 ( ,  등)
-   패턴 체크
- . 파일 . 등록 확인

### . -   구현됨

**목적** 학습 데이터 자동 수집

**수집 대상**
-  콘텐츠 성과 데이터
-  프롬프트/응답 패턴
- 크루즈 도메인 고객  
- 일일 실행 통계

### . -   구현됨

**목적** 병렬 실행 시 리소스 관리

**기능**
- /메모리 사용률 실시간 체크
- 리소스 상태에 따른 배치 크기 동적 조절
-  에이전트 자동 감지

## 주의사항

-  스크립트는 **가볍게** 유지 (실행 시간  초)
- 장시간 작업은 백그라운드로 실행
- 에러 발생 시 이 에이전트 실행을 차단하지 않도록 주의
-  환경에서는   사용 권장
