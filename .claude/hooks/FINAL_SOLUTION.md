#   시스템 최종 솔루션

##  문제 해결 완료

**문제**  의 자동  트리거가 작동하지 않음
-  # 에서  % 실패
-  # 환경 변수 전달 안 됨

**해결** 수동  시스템 완성 → **더 강력하고 유연함**

---

##  지금 바로 사용하기

### ⃣ 가장 간단한 방법


# - 실행
 .//-.  에이전트-이름

#  로 에이전트 실행 ( 에서)

# - 실행
 .//-.  에이전트-이름


### ⃣ 실전 예시

####  개발

 .//-.  -
# →  "-로  // 만들어줘"
 .//-.  -


#### 크루즈 도메인

 .//-.  -
# →  "수수료 계산 로직 만들어줘"
 .//-.  -


####  콘텐츠 생산

 .//-.  --
# →  "유튜브 콘텐츠 개 자동 생산"
 .//-.  --

###  자동 생산

 .//-/-.

##  자동으로 실행되는 기능

### - (실행 전)

####  모든 에이전트 ()
-  시작 로그 기록
-  환경 변수 체크 (.)
-   브랜치 확인
-  컨텍스트 크기 측정

####    에이전트
-  **  자동 로드** (에이전트별 맞춤 섹션)
  - - → 수수료/정산 섹션
  - - → /예약 실수 섹션

####   
-  **  할당량 체크**
-  **/메모리 리소스 관리**
-  **배치 크기 동적 조절** ( →  →  → )
-  학습 데이터 로드 (성과 패턴)

####   에이전트 (-, - 등)
-  **  자동 로드** (토큰 % 절감)
-  관련 스킬 로드 (--, -- 등)
-   변경 사항 확인

####   에이전트
-  **환경 변수 체크** ()
-  ** 스키마 검증**
-  위험 작업 경고 (마이그레이션 등)

####   (/ 작업 시)
-  **하드코딩된 시크릿 탐지** ( ,  등)
-  **  패턴 체크**
-  .  확인

### - (실행 후)

####  모든 에이전트 ()
-  완료 로그 기록
-  성능 메트릭 저장 ()

####   에이전트
-  ** 타입 체크 자동 실행**
-  ** 자동 실행**
-  품질 리포트 저장

####    (모든 코딩 작업)
-  **함수 복잡도 체크** (줄 이하)
-  **테스트 커버리지 체크** (% 권장)
-  ** 경고 체크** (개 미만 권장)
-  **종합 점수 계산** (/ 기준 /)

####   (//)
-  **학습 데이터 자동 수집**
  -  성과 데이터 (조회수, 참여율 등)
  -  고객  
  -  프롬프트/응답 패턴
-  일일 실행 통계 집계

├── /                   # 병렬 실행 리소스 로그
└── /                   # 보안 검사 리포트
