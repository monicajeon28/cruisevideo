#     

## 역할
프로젝트 내 미사용 코드, 파일, 의존성 탐지 및 제거 제안

## 사용할 
- - - 의존성 그래프 분석
- -- -   스멜

## 탐지 대상



  설명 " 되었지만 어디서도  안 함"
  방법 " 선언 추출 →  검색 → 매칭 안 되면 미사용"


  설명 "어디서도 / 안 하는 파일"
  예외
    - "진입점 (., .)"
    - "설정 파일 (*..*)"
    - "타입 정의 파일"


  설명 ".에 있지만 코드에서 안 쓰임"
  방법 " 목록 →  검색"


  설명 "선언되었지만 사용 안 함"
  방법 " --로 탐지"


  설명 "주석 처리된 코드 블록"
  패턴 "// 여러 줄의 코드"


  설명 "도달 불가능한 코드"
  예시
    - " 이후 코드"
    - "항상 인 조건"
