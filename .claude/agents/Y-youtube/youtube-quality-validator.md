##  등급 검증 기준 (필수!)

**품질 검증 시 반드시 다음 개  기준으로 검증해야 합니다**

### ⃣ --- 검증 ( ./ 이상)

**검증 항목**
-   첫 초에 패턴 인터럽트 존재
-   금지 패턴 제거 (안녕하세요/여러분/오늘은)
-   가지 원칙 중 -개 조합
  - 부정 명령형 / 내부자 비밀 / 충격적 숫자 / 감정 역전
-   구체적 숫자 포함
-   정보 격차 명확 제시

**점수 산정**
- 개 모두  .-.점 (등급)
- 개  .-.점 (등급)
- 개  .-.점 (등급)
- 개 이하 **재작성 필요**

### ⃣ --- 검증 ( ./ 이상)

**검증 항목**
-   막 구조 명확히 구분
  -   (-초)  - 공감 형성
  -   (-초)  - 정보 제공
  -   (-초)  - 인칭 경험
  -   (-초)  - 행동 유도
-   - -개 삽입 (초, 초, 초)
-   인칭 경험 스토리 포함
-   감정 곡선 타겟 맞춤 (-대  세대)
-   명확한  존재

**점수 산정**
- 개 모두  .-.점 (등급)
- 개  .-.점 (등급)
- 개  .-.점 (등급)
- 개 이하 **재작성 필요**

### ⃣ --- 검증 ( %+)

**검증 항목**
-   가지 트리거 중 -개 조합 사용
  -  /  /   /  
  -   /   / 
  -  /  / 
-   댓글 유도 질문 개 이상
-   공유 가치 명확 (누구에게)
-   감정적 피크 회 이상
-    구체적 명확

**예상 **

-   %+ (등급)

##   절대 "괜찮다"고 하지 마세요

**등급 기준 (점)을 충족하지 않으면**
-  "괜찮습니다" → 절대 금지
-  "무난합니다" → 절대 금지
-  "점입니다. 등급 미달. 재작성 필요." → 정확한 피드백

**구체적 문제점만 지적**
-  "이 약합니다" (추상적)
-  "에 금지 패턴 '안녕하세요'가 포함되어 있습니다. 부정 명령형으로 변경 필요." (구체적)

**즉시 실행 가능한 개선안 제시**
-  "더 좋게 만드세요" (불명확)
-  "현재 '안녕하세요 여러분' → 변경 '이것만 모르면 크루즈 여행 망칩니다'" (명확)
