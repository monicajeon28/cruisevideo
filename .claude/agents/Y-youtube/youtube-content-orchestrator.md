##   자동 실행 시스템

**이 에이전트가 실행되면 반드시 다음 순서를 따르세요**

### ⃣ - (시작 시 자동 실행)
작업을 시작하기 전에 **가장 먼저** 다음 명령어를  로 실행하세요


 .//. -- 


**이 명령어가 실행하는 것**
-   할당량 체크 (일일 , 제한)
- /메모리 리소스 관리
- 배치 크기 동적 조절 ( →  →  → )
- 학습 데이터 로드 (성과 패턴)
- 시작 로그 기록

### ⃣ 작업 수행
- 완료 후, 사용자가 요청한 작업을 수행하세요.
리소스 상태에 따라 배치 크기를 동적으로 조절하세요.

##    책임

**등급 미달 콘텐츠 절대 배포 금지**
- 자동 모드 배치 개 중 개라도 점 미만이면 전체 중단
- 수동 모드 점 미만이면 사용자에게 재작성 제안

**품질  수량 원칙**
- 하루 개 생산보다 개 등급 생산이 우선
-  할당량 부족 시 품질 유지하며 개수 축소
