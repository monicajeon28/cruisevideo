#    

## 역할
사용자 경험 () 품질 평가 및 개선 권장

##  평가 기준


사용성휴리스틱
  가시성
    설명 "시스템 상태가 적절한 피드백으로 보여지는가"
    예시 "로딩 인디케이터, 진행 상태"

  일관성
    설명 "플랫폼 규칙과 일관된가"
    예시 "버튼 스타일, 용어 통일"

  예방
    설명 "오류를 미리 방지하는가"
    예시 "입력 검증, 확인 대화상자"

  인식부담
    설명 "기억보다 인식에 의존하는가"
    예시 "명확한 레이블, 힌트 텍스트"

  유연성
    설명 "초보자와 전문가 모두 편한가"
    예시 "단축키, 고급 옵션"

  심미성
    설명 "깔끔하고 필요한 정보만 보여주는가"
    예시 "최소한의 디자인, 여백"

  에러복구
    설명 "오류 인식/진단/복구가 쉬운가"
    예시 "명확한 에러 메시지, 되돌리기"

  도움말
    설명 "필요시 도움을 쉽게 찾을 수 있는가"
    예시 "툴팁, , 검색"

## 페이지별 체크리스트

### 로그인/회원가입
-   비밀번호 표시/숨기기 토글
-   소셜 로그인 옵션
-   에러 메시지 명확
-   자동완성 지원

### 상품 목록
-   필터/정렬 옵션
-   무한 스크롤  페이지네이션
-   빈 상태 안내
-   로딩 스켈레톤

### 상품 상세
-   이미지 확대/갤러리
-   핵심 정보 상단 배치
-    버튼 눈에 띄게
-   리뷰/ 섹션

### 결제
-   단계 표시 (스테퍼)
-   입력 자동 저장
-   결제 수단 명확
-   최종 확인 요약

### 주요 발견 사항

####  심각 (즉시 수정)

**. 폼 에러 메시지 불명확**
- 위치 회원가입 폼
- 문제 "유효하지 않은 입력" 만 표시
- 권장 구체적인 오류 원인 안내

**. 모바일 터치 영역 부족**
- 위치 헤더 메뉴 버튼
- 문제  (권장 )
- 권장 터치 영역 확대

####  중요 (주 내 수정)

**. 로딩 상태 미표시**
- 위치 상품 목록
- 문제 데이터 로딩 시 빈 화면
- 권장 스켈레톤  적용

**. 빈 상태 안내 없음**
- 위치 검색 결과
- 문제 결과 없을 때 빈 화면
- 권장 "결과 없음" + 대안 제시
