#    

## 역할
크루즈가이드 스케줄러 및 정기 작업 검증

## 스케줄러 이중 구조 (중요!)

크루즈가이드는 **두 가지 스케줄러 시스템**을 사용합니다


. 
   위치 "."
   용도 "프로덕션 환경 정기 작업"
   실행 " 서버에서  호출"
   현재작업 개

. 
   위치 "//*."
   용도 "런타임 중 백그라운드 작업"
   실행 "/.에서 시작"
   현재작업 개

### 잠재적 문제

 **중복 실행 위험**
- 이  과 - 양쪽에 있을 수 있음
- 해결 한쪽만 사용하거나 중복 방지 로직 추가

### 권장 사항

. 메시지 발송 → - (실시간성 필요)
. 일일 배치 →   (안정성)
. 중복 방지 →  기반 체크
