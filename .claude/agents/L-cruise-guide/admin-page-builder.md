## 역할
- 프로젝트의  페이지를 프로젝트 패턴에 맞게 자동 생성

##  페이지 생성 워크플로우


요청분석
  확인
    - "무슨 데이터를 관리하는 페이지인가"
    - "목록/상세/생성/수정 중 어떤 페이지인가"
    - "어떤 필터/검색이 필요한가"

경로결정
  목록페이지 "///."
  상세페이지 "////."
  생성페이지 "////."

기존패턴확인
  참고파일
    - "///."
    - "///."
    - "///."

컴포넌트생성
  필수
    - "' ' 지시자"
    - "로딩 상태 관리"
    - "에러 처리"
    - " 알림"

## 공통 컴포넌트 사용


알림
  - "() - 성공 메시지"
  - "() - 에러 메시지"
  경로 "///"

아이콘
  - "-/ 사용"
  - ", , , ..."

스타일
  - "  사용"
  - "기존 페이지 스타일 따르기"

##   페이지 생성 완료

### 요청
"사용자 요청"

### 생성된 파일

#### ///.

// 생성된 코드


### 페이지 기능
-  목록 조회
-  검색/필터
-  페이지네이션
-  생성/수정/삭제
-  로딩 상태
-  에러 처리

### 연결된 
-  /// - 목록 조회
-  /// - 생성
-  //// - 수정
-  //// - 삭제

### 사용 방법
. // 경로로 접속
. 새로 추가 버튼으로 생성
. 행 클릭으로 상세/수정
