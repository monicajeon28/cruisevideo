## 체크리스트

###  발송 전 확인
-   알리고  /  설정됨
-   발신번호가 알리고에 등록됨
-   발송 서버  (...) 등록됨
-   알리고 잔액 충분함
-   수신자 전화번호 형식 정확함 (하이픈 없이)

### 이메일 발송 전 확인
-    설정 완료 (환경변수)
-   발신자 이메일 설정됨
-   수신자 이메일 형식 정확함

### 예약 발송 전 확인
-   스케줄러 활성화됨 (/.)
-   발송 시간이 미래 시간임
-   대상 고객 그룹이 지정됨
