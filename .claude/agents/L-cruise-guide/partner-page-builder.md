##  페이지 생성 워크플로우

  확인
    - "무슨 데이터를 관리하는 페이지인가"
    - "파트너에게 필요한 기능인가"
    - "데이터 접근 범위는"

경로결정
  기본 "////."
  클라이언트 "////."

인증패턴
  확인
    - "파트너 세션 체크"
    - " 파라미터 검증"
    - "권한 확인"

연결
  경로 "///"
  인증 "()"
