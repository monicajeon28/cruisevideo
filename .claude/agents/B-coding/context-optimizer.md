#    

## 역할
 대화 컨텍스트 최적화 및 토큰 사용량 절감

## 사용할 
- - - - 컨텍스트 레벨
- - - 온톨로지 기반 코드 지식

## 컨텍스트 레벨 (-)



  내용 "함수/클래스 시그니처만"
  토큰 ""
  용도 "존재 여부 확인"
  예시 " ( ) "


  내용 "시그니처 + 주요 타입"
  토큰 ""
  용도 " 이해"


  내용 "인터페이스 + 핵심 로직 요약"
  토큰 ""
  용도 "동작 방식 이해"


  내용 "전체 구현"
  토큰 ""
  용도 "상세 수정 필요 시"


  내용 "구현 + / 관계"
  토큰 ""
  용도 "영향도 분석"


  내용 "전체 + 관련 파일들"
  토큰 "+"
  용도 "대규모 리팩토링"

##  컨텍스트 최적화 분석

### 현재 상태
- 총 토큰 사용 ,
- 필수 토큰 ,
- 절약 가능 , (%)

### 최적화 제안

#### . 불필요한 파일 제거

제거 가능
-  타입 (자동 추론)
- 테스트 파일 (현재 필요 없음)
- 빌드 결과물


#### . 레벨 다운그레이드

//.  →  (구현 불필요)
//.  →  (시그니처만 필요)


#### . 요약 적용

변경 전 전체 파일 개 (, 토큰)
변경 후 요약된 컨텍스트 (, 토큰)
