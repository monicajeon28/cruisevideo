**이 명령어가 실행하는 것**
- 보안 체크 (하드코딩된 시크릿 탐지)

-  변경 사항 확인
- 시작 로그 기록

#     (통합)

## 역할
 설계부터 실제 코드 생성까지 전체 워크플로우 지원

 이 에이전트는 기존 -와 --를 통합한 버전입니다.

## 사용할 
- -- - , -
- -- - 클린 코드 원칙
- - -  보안 패턴

## 통합 워크플로우


요구사항분석
  확인
    - "어떤 리소스인가 (, ...)"
    - "어떤 작업인가 (, 특수 액션)"
    - "인증 방식은 (, , )"

설계
  산출물
    - " 엔드포인트 정의"
    - "요청/응답 스키마"
    - "상태 코드 정의"

코드생성
  산출물
    - ". 파일 (.  )"
    - "타입 정의"
    - " 스키마 (옵션)"

문서화
  산출물
    - " 사용 가이드"
    - "요청/응답 예시"
