#   

## 역할
여러 에이전트를 조합하여 복잡한 작업 수행

## 에이전트 체인 패턴


순차체인
  설명 "하나씩 순서대로 실행"
  예시 "코드작성 → 리뷰 → 테스트 → 커밋"

병렬체인
  설명 "동시에 여러 검사"
  예시 "보안검사 + 성능검사 + 타입검사"

조건부체인
  설명 "결과에 따라 다음 에이전트 결정"
  예시 "리뷰결과 등급 → 리팩토링 에이전트"


## 작업별 에이전트 조합

### . 새 기능 개발


워크플로우 "-"
단계
  기획
    에이전트 "--"
    입력 "기능 요구사항"
    출력 "기술 명세서"

  설계
    에이전트 "-"
    입력 "기술 명세서"
    출력 "아키텍처 다이어그램"

  구현
    병렬
      - "-" " 설계"
      - "-" "타입 정의"

  리뷰
    에이전트 "-"
    조건
      "-등급" "다음 단계"
      "이하" "- 실행"

  테스트
    에이전트 "-"
    조건
      "커버리지 %+" "다음 단계"
      "미달" "테스트 추가"

  완료
    에이전트 "-"
    출력 " 생성"


### . 버그 수정


워크플로우 "-"
단계
  분석
    에이전트 "-"
    병렬
      - "-" "로그 분석"
      - "-" "영향 범위"

  수정
    에이전트 "--"
    입력 "버그 위치, 원인"

  검증
    병렬
      - "-" "테스트"
      - "-" "보안 점검"

  커밋
    에이전트 "-"
    메시지 " 버그 설명"


### . 코드 리뷰


워크플로우 "-"
단계
  자동분석
    병렬
      - "-" "종합 리뷰"
      - "-" "보안"
      - "-" "성능"
      - "-" "코드 스멜"

  집계
    형식 
      ## 코드 리뷰 결과

       영역  등급  이슈 
      ------------------
       품질     
       보안     
       성능     

  후속
    조건부
      "보안 이슈" "- 상세 분석"
      "성능 이슈" "- 또는 -"


### . 프로덕션 배포


워크플로우 "-"
단계
  사전점검
    병렬
      - "-" "배포 체크리스트"
      - "-" "환경변수"
      - "--" "린트/타입/테스트"

  영향분석
    에이전트 "-"
    입력 "변경된 파일 목록"

  성능확인
    병렬
      - "-" "번들 크기"
      - "-" "  "

  최종승인
    체크리스트
      - "모든 검사 통과"
      - "리뷰어 승인"
      - "테스트 %"
