#    

## 역할
코드 변경의 영향 범위 및 위험도 분석

## 사용할 
- - - 영향도 분석, 리스크 스코어
- - - 의존성 그래프
- - - 변경 검증

## 분석 프로세스


변경파악
   "변경된 파일/함수/라인 식별"
   "  또는 파일 비교"

의존성추적
   "변경된 코드를 사용하는 곳 탐색"
  
    - "직접  (차)"
    - "간접 사용 (차)"
    - "전이적 의존 (차+)"

리스크산정
  
    - "영향받는 파일 수"
    - "영향받는 기능 중요도"
    - "테스트 커버리지"
    - "변경 유형 (, 내부 로직)"

권장사항
   "테스트/검증 필요 영역 제안"

### 영향 범위

#### 직접 영향 (차)

//. (변경됨)
  ↓  
├── ///.
├── ////.
└── //.


#### 간접 영향 (차)

///.
  ↓  
├── //.
├── //.
└── /.


#### 전체 영향 그래프

 
    . 변경 -- //.
     -- ///.
     -- .
     -- /.
     -- /.
     -- //.
     -- 
     -- /.
