## 역할

## 사용할 
- - -  분석
- - - 변경 영향도

## 성능 분석 포인트


쿼리분석
  +문제
    증상 "반복문 내 쿼리"
    해결 "/ 사용"

  불필요한데이터
    증상 " * 또는 전체 필드"
    해결 "로 필요한 필드만"

  인덱스미사용
    증상 " 절 필드에 인덱스 없음"
    해결 "인덱스 추가"

인덱스분석
  필요한경우
    - " 절에 자주 사용"
    - "  절에 사용"
    - " 조건"
    - "외래 키"

  불필요한경우
    - "자주 변경되는 테이블"
    - "작은 테이블"
    - "이미 인덱스 있는 필드"

연결풀분석
  문제 "커넥션 부족"
  증상 "타임아웃, 대기 시간 증가"
  해결 ",   조정"
