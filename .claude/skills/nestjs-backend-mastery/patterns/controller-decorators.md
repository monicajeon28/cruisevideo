#   

  에서 필수 데코레이터와 타입 안전성 가이드

## 핵심 원칙



  목표 "웹/모바일에서 를 가져다 쓸 수 있도록 완전하게"
  이유
    - 타입/ 미추출 시   사용
    - 수정 시 유지보수  ( +  + )
    - 로 문제 캐치 불가 → 런타임 버그


---

## . 필수 데코레이터 체크리스트

###  레벨

 데코레이터  필수  용도 
----------------------
 ('')    라우트 경로 
 ('')     그룹 (복수형) 
 ('-')  조건부  인증 필요 시 
 ()  조건부  인증 가드 

###  레벨

 데코레이터  필수  용도 
----------------------
 ///     메서드 
 (  )     설명 
 ( ,  )    모든 응답 타입 
 ()  조건부   파라미터 
 ()  조건부   파라미터 
 (  )  조건부    
 (.*)  조건부  상태 코드 지정 

###  레벨

 데코레이터  필수  용도 
----------------------
 ()  조건부    
 ('', )  조건부   파라미터 + 파이프 
 ()  조건부   파라미터

##  작성 시 체크리스트

###  레벨
-   ('복수형') 적용
-   () 인증 필요시 적용
-   () 가드 적용

### 각 
-   (  ) 작성
-   ( ,  ) 모든 응답 코드 정의
-   ()  파라미터 문서화
-   ()  파라미터 문서화
-   (  )   타입 명시
-   반환 타입  명시
-   로 런타임 검증

### 
-    는 -로 정의
-    는 () 포함
-   민감 정보 () 처리
