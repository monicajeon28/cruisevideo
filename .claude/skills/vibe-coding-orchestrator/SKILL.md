#    .

**개  스킬을 자동으로 감지, 호출, 조율하는 마스터 오케스트레이터**

## . 업데이트

### 신규 스킬 -   
- **--** 바이브 코딩 → 프로덕션 전환 가이드
  - 단계 성숙도 모델 ( →  →  → )
  - / →  마이그레이션 경로
  - 유료 서비스 필수 기능 (, 사용량 관리)
  - 비용 최적화 전략
  -  ()  가이드

### . 스킬 (유지)
- **-** 프로젝트 구조 설계 (레이어 분리, 기능 모듈화)
- **--**  현대적 기술 스택 선택 가이드
- **-** 요구사항 → 기술 스펙 변환
- **--** 네이밍 규칙 자동 적용
- **--** 가지 코드 스멜 탐지

###  현대 도구 통합
- ****  대체 (-% 빠름)
- **/**  대체 (-배 빠름)
- **/** 상태관리 현대화
- **  ** 서버 상태 필수
- ****  →  클라이언트 자동생성
- **   + ** 폼 검증 표준
- ** + ** 모노레포 표준
- ** ()**   권장 (  최적화)

## 스킬 레지스트리 (개)

     설명 
---------------------
 ****  -, --, -  시작 전 설계 
 ****  -, -  코드베이스 분석 
 ****  -, -  변경 영향도 
 ****  --, -, -  코드 품질 
 ****  -, --  구조 설계 
 ****  --, --  검증 
 ****  -  최종 리뷰 
 ****  --  프로덕션 전환
