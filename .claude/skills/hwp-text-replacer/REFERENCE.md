#    

**. 상용 수준  텍스트 치환 엔진**

길이가 다른 텍스트로의 치환을 완벽하게 지원하는  문서 처리 스킬입니다.

## 핵심 발견    플래그


  (바이트)
┌──────────┬────────────────────────┐
│     │  -              │
│ 컨트롤   │ 실제 문자 수           │
│ 플래그   │                        │
└──────────┴────────────────────────┘


**중요**   플래그를 보존하지 않으면 파일 손상!

## 사용 예시


 ..  

# 기본 사용
  (".")
.("이름", "홍길동")
.("주소", "서울시 강남구 테헤란로 ")
.(".")

# 다중 치환
  .(
    "이름" "홍길동",
    "주소" "서울시 강남구",
    "날짜" "년 월 일"
)
