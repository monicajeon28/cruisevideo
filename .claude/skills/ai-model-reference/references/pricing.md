##  비용 최적화 전략

### . 캐싱 활용 (최대 % 절감)

시스템 프롬프트 + 고정 컨텍스트 → 캐싱
사용자 입력 + 동적 데이터 → 일반 처리

예시 , 요청, % 캐시 히트 ( )
- 캐싱 없음 $,
- 캐싱 적용 $, (% 절감)


### . 배치 처리 (% 절감)

실시간 불필요한 작업
- 데이터 분석
- 콘텐츠 생성
- 벌크 처리

시간 내 처리 → % 할인
