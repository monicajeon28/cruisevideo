 모델  호출 시 정확한 모델명과 가격 정보를 빠르게 참조할 수 있는 가이드입니다.

##    - 즉시 사용 가능한 모델명

###  추론 모델 (복잡한 문제 해결)
 제공사  모델   호출명    / (/) 
-----------------------------------------------------
     ---    $. / $. 
   -  ----    $. / $. 
     -    $. / $. 
       ---  ****  $.$. / $.$. 
    .   -.-  ****  $. / $. 
    .  ----    $. / $. 

###   모델 (일반 작업)
 제공사  모델   호출명    / (/) 
-----------------------------------------------------
   -.  -.    $. / $. 
   -  ----    $. / $. 
   - 최신  ---    $. / $. 
   -.  -.---  ****  $. / $. 
    .  ----    $. / $. 
      ---    $. / $. 
    .   -.-  ****  $. / $.$. 

###  가성비 모델 (대량 처리/저비용)

-----------------------------------------------------
   -   --    $. / $. 
   -   --    $. / $. 
   -.   -.----  ****  $. / $. 
    . -  -.--  ****  $. / $. 
    . -  -.--  ****  $. / $. 
      ---    $. / $. 
     -    $. / $.

### 상세 정보 참조
- 전체 모델 목록 및  호출명 /.
- 상세 가격 및 캐싱 비용 /.

## 빠른 선택 가이드

### 복잡한 추론/코딩 작업

 , -
 ----, ---
 ---, -.-
 -


### 빠른 응답이 필요한 일반 작업

 -., -, -
 ----, ---
 -.-


### 대량 처리/비용 최적화

 -- ($./$.)
 ---- ($./$.)
 -.-- ($./$.)
 - (- % 할인)


## 비용 절감 전략

### . 프롬프트 캐싱 (% 절감 가능)
- ****   .,   . (% 절감)
- ****   $./ (- 기준 % 절감)
- ****   %   

### . 배치 처리 (% 절감)
- 시간 내 비동기 처리로 입출력 % 할인
-   ,    지원

### . 모델 계층화 전략

간단한 작업 → / (저비용)
     ↓ 복잡도 증가 시
중간 작업 → / (균형)
     ↓ 복잡도 증가 시
복잡한 작업 → / (고성능)
