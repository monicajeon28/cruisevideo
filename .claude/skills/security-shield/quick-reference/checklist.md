##   

###  


-   코드에  키, 비밀번호, 토큰 없음
-   환경변수 또는   사용
-   . 파일이 .에 있음
-   ..은 값 없이 키만


###  


-   모든 사용자 입력에 검증 있음
-   에 - 데코레이터
-   은   사용
-   파일 업로드 타입/크기 제한


### /


-   모든 보호 엔드포인트에  적용
-   역할 기반 접근 제어() 구현
-   비밀번호는 /로 해시
-    만료 시간 적절 ( 분,  일)


### / 


-    사용 안 함
-   사용자 입력을  에 넣지 않음
-    출력 시 이스케이프
-   ()에 사용자 입력 없음


###  


-   에러 메시지에 민감 정보 없음
-   스택 트레이스가 프로덕션에 노출 안 됨
-   인증 실패 시 일관된 메시지


## - 

###  


-    미들웨어 적용
-   -- 설정
-   --- 
-   -- 
-    올바르게 설정


###  & 


-     만료 분 이하
-     만료 일 이하
-    은  쿠키
-     적용
-   로그아웃 시 토큰 무효화


###  


-   로그인   
-   회원가입   
-   일반   


### 


-     통과
-   취약한 패키지 업데이트
-   불필요한 의존성 제거


### 


-   보안 이벤트 로깅
-   민감 정보 로그에 기록 안 함
-   알림 시스템 구축
