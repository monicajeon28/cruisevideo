## 코드 작성 시 필수 확인

### 시크릿 관리
-   코드에  키, 비밀번호, 토큰 없음
-   환경변수 또는   사용
-   . 파일이 .에 있음

### 입력 검증
-   모든 사용자 입력에 검증 있음
-   에 - 데코레이터
-   은   사용

### 인증/인가
-   모든 보호 엔드포인트에  적용
-   역할 기반 접근 제어() 구현
-   비밀번호는 /로 해시


## 문서 구조


-/
├── .                          # 이 파일 (라우터)
├── /
│   ├── -.          # + 시크릿 탐지 패턴
│   └── -.             #  고급 패턴
├── /
│   ├── -.        # ., , 
│   ├── -.            # , , 
│   ├── -.                #  보안 패턴
│   └── -.              #  
├── /
│   ├── -.                #  인증 템플릿
│   └── -.             #  보안 템플릿
├── /
│   ├── -.               #  보안 패턴
│   └── .                 # 입력 검증 패턴
└── -/
    └── .                  # 보안 체크리스트
