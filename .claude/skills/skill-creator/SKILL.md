### .   (점진적 공개)

**스킬은 매뉴얼처럼 작동합니다** - 목차 → 장 → 상세 부록 순서로 필요할 때만 로드


   +  (시작 시 시스템 프롬프트에 로드,  토큰)
  . 전체 (관련성 판단 시 로드,  토큰)
  참조 파일들 (필요 시에만 개별 로드)


**실전 적용**
- .는 줄 이하 유지

## 스킬 유효성 체크리스트

### 핵심 품질
-   이 구체적이고 키워드 포함
-   이 "무엇을" + "언제" 모두 포함
-   . 줄 이하
-   상세 내용은 별도 파일로 분리
-   시간 의존 정보 없음
-   일관된 용어 사용
-   파일 참조는 단계만
-   인칭으로 작성

### 코드 & 스크립트
-   스크립트가 에러를 직접 처리 (에 위임 )
-   매직 넘버 없음 (모든 값에 이유 설명)
-   필요 패키지 목록화
-    경로 없음 (모두 / 사용)
-   검증/확인 단계 포함

### 테스트
-   최소 개 평가 시나리오 생성
-   , , 에서 테스트
-   실제 사용 시나리오로 테스트
