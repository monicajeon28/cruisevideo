##  가지 공유 트리거 (효과 순)

### ⃣  (   ) - 효과 %

**패턴** 시간 제한 + 희소성 + 타인 경험

**크루즈 예시**
-  "믿을 수 없지만 저는 년을 기다렸어요..."
-  "이 가격은 다음 주까지만 가능합니다"
-  "지금 자리만 남았다고 하더라고요"

**사용법** 긴급성 + 희소성 → 즉각적 행동 유도

---

### ⃣  (희소성) - 효과 %

**패턴** 한정 + 수량 + 긴급성

**크루즈 예시**
-  "이 경험은 단 자리만 남았어요"
-  "시즌 마지막 크루즈입니다"
-  "년에 딱 번만 운항하는 코스"

**사용법** 구체적 수량 명시 → 희소성 강조

---

### ⃣   (사회적 증거) - 효과 %

**패턴** 대규모 집단 + 통계 + 검증

**크루즈 예시**
-  "우리 그룹 ,명이 발견한 것"
-  "작년 만 명이 선택한 코스"
-  "시니어 여행 위 크루즈"

**사용법** 대규모 집단 → 신뢰도 증폭

---

### ⃣   (감정적 기쁨) - 효과 %

**패턴** 놀라움 + 행복 + 감동

**크루즈 예시**
-  "부모님 반응을 보면 깜짝 놀랄 거예요"
-  "평생 못 잊을 추억이 되었습니다"
-  "눈물 나올 뻔했어요"

**사용법** 감정 피크 → 공유 충동 유발

---

### ⃣   (럭셔리 어필) - 효과 %

**패턴** 가격 대비 가치 + 프리미엄 경험

**크루즈 예시**
-  "만원짜리인 줄 알았는데..."
-  "성급 호텔보다 저렴합니다"
-  "이 가격에 이런 럭셔리"

**사용법** 가격  가치 갭 → 놀라움 유발

---

### ⃣   (패턴 인터럽트) - 효과 %

**패턴** 상식 뒤집기 + 반전

**크루즈 예시**
-  "대부분 크루즈 객실이 작다고 생각하지만..."
-  "크루즈가 비싸다는 건 오해입니다"
-  "사실 육지 여행보다 저렴해요"

**사용법** 기존 믿음 뒤집기 → 공유 욕구 증폭

---

### ⃣  (권위) - 효과 %

**패턴** 전문가 + 경력 + 내부자 정보

**크루즈 예시**
-  "세 크루즈 전문가가 밝힌 비결"
-  "년 차 승무원이 알려주는 꿀팁"
-  "여행사 대표가 절대 안 알려주는 것"

**사용법** 전문성 강조 → 신뢰도 구축

---

### ⃣  (공감 가능성) - 효과 %

**패턴** 나도 그랬어 + 공통 경험

**크루즈 예시**
-  "당신도 객실에 갇힌 느낌 받았나요"
-  "저도 처음엔 멀미 걱정했어요"
-  "혼자 가기 망설여지셨죠"

**사용법** 공통 경험 → 유대감 형성

---

### ⃣  (모순/역설) - 효과 %

**패턴** 예상 반대 + 놀라운 결과

**크루즈 예시**
-  "크루즈 끊은 후 오히려 돈이 늘었어요"
-  "더 비싼 객실이 사실 저렴합니다"
-  "많이 먹을수록 건강해졌어요"

**사용법** 역설 → 호기심 증폭

---

###   (친밀감) - 효과 %

**패턴** 친구/가족 + 개인적 추천

**크루즈 예시**
-  "제 친한 친구(세)가 최고의 경험이라고..."
-  "부모님이 또 가고 싶다고 하세요"
-  "우리 동네 명이 함께 갔어요"

**사용법** 가까운 관계 → 개인적 추천 효과
