-     시나리오를 파악했는가

-   테스트 데이터를 준비했는가
-   이 필요한 의존성을 파악했는가

### 기본 검사
-   함수명이 동작을 명확히 설명하는가
-   파라미터 수가 개 이하인가
-   함수 길이가 줄 이하인가
-   단일 책임 원칙을 따르는가

### 타입 검사
-   모든 파라미터에 타입이 있는가
-   반환 타입이 명시되어 있는가
-    타입을 사용하지 않았는가

### 에러 처리
-   예외 상황이 처리되었는가
-   / 체크가 있는가

### 테스트
-   테스트가 작성되었는가
-   엣지 케이스가 커버되는가
