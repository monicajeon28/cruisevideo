## 가 낮을 때 점검 사항

###    %
-   실행되지 않는 코드 경로가 있는가
-   에러 핸들링 코드가 테스트되지 않았는가
-   조건문의 모든 분기가 테스트되었는가

###    %
-   -의 모든 분기 테스트
-   -의 모든 케이스 테스트
-   삼항 연산자의 양쪽 테스트
-    의  케이스 테스트

###    %
-    메서드가  메서드를 통해 테스트되는가
-   콜백 함수가 테스트되는가
-   에러 핸들러 함수가 테스트되는가
