#  데이터베이스 직접 연결 

  데이터베이스에 직접 연결하여 테이블 생성,  실행 등의 작업을 수행하는 입니다.

## 사용 시점
-  테이블 생성/수정/삭제 필요 시
-  쿼리 직접 실행 필요 시
- 데이터베이스 스키마 확인 필요 시
- 데이터 마이그레이션 필요 시

## 연결 정보 확인

### 단계 프로젝트 . 파일 확인

# 현재 작업 중인 프로젝트 폴더에서 . 파일 읽기
 .   


필요한 환경변수
-   호스트 (예 ----...)
-  포트 ( 또는 )
-  사용자명 (예 .)
-  데이터베이스 비밀번호

### 단계 .에 정보가 없을 경우
사용자에게 다음 정보 요청

 연결 정보가 필요합니다
.           ()
.
