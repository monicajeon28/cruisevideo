## 새 코드 위치 결정

### 질문 체크리스트
. 여러 앱에서 사용되는가
   -  → 번으로
   -  → 해당 앱 내부

. 플랫폼 독립적인가
   -  → /
   -  ( 필요) → / 또는 앱 내부

.  컴포넌트인가
   -  + 공유 → /
   -  + 앱 전용 → 해당 앱의 /

. 타입/인터페이스인가
   - 공유 → ////
   - 앱 전용 → 해당 앱의 /

. 유틸리티 함수인가
   - 공유 → ////
   - 앱 전용 → 해당 앱의 /
