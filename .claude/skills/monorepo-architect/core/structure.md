새 코드 작성 시

. 여러 앱에서 사용
   ├──  → 플랫폼 독립적
   │         ├──  → /
   │         └──  () → /
   └──  → 해당 앱 내부

   ├── 공유 → ////
   └── 앱 전용 → 해당 앱의 /

   ├── 공유 → ////
   └── 앱 전용 → 해당 앱의 /
