#  등급   - -초 최적화

## 등급 달성 목표


 "% 이상 (초 이탈률 % 이하)"
 "./ 이상"
 "% 이상"


**중요** 등급 은 일반  학습 패턴이 아닌, **실제 고성과 영상 데이터 기반**입니다.

---

##  가지 핵심 원칙

### ⃣ 부정 명령형 ( )

**효과**  +%, 시선 고정 .배 증가

**패턴**
- 절대 하지 마세요
- 이것만 모르면
- 이거 안 하면

**크루즈 예시**
-  "이것만 모르면 크루즈 여행 망칩니다"
-  "크루즈 타기 전 절대 하지 마세요"
-  "이 실수만 안 하면 크루즈 완벽합니다"

**절대 금지**
-  "안녕하세요, 여러분"
-  "오늘은 에 대해 알려드리겠습니다"
-  "지금부터 을 시작하겠습니다"

---

### ⃣ 내부자 비밀 ( )

**효과** 공유율 +%, 댓글 유도 +%

**패턴**
- 직원만 아는
- 승무원이 말 못 하는
- 크루즈 회사가 숨기는

**크루즈 예시**
-  "승무원들만 아는 크루즈 무제한 먹방 비법"
-  "크루즈 회사가 절대 말 안 하는 객실 등급 차이"
-  "년 차 승무원이 밝힌 숨겨진 공간"

**핵심** 정보 격차( ) 유발 → 호기심 극대화

---

### ⃣ 충격적인 숫자 ( )

**효과** 시청 지속 +%, 완주율 %

**패턴**
- 예상과 다른 가격
- 놀라운 포함 항목
- 숨겨진 비용

**크루즈 예시**
-  "크루즈 박 일 만원 진짜 전부 포함입니다"
-  "박에 만원인데 식사 무제한 믿기지 않는 가격"
-  "수천만원 아닙니다. 실제론 이 가격입니다"

**핵심** 인지 부조화 → 시청 지속 유도

---

### ⃣ 감정 역전 ( )

**효과** 공감도 +%, 공유 충동 +%

**패턴**
- 처음엔 걱정했지만
- 실패인 줄 알았는데
- 후회할 뻔했어요

**크루즈 예시**
-  "처음엔 크루즈가 비싸다고 생각했어요. 그런데..."
-  "솔직히 실망할 줄 알았습니다. 근데 완전히 달랐어요"
-  "부모님 모시고 크루즈 걱정 많았죠. 결과는..."

**핵심** 부정→긍정 전환으로 감정 몰입

##  등급 체크리스트

을 작성했다면 반드시 확인

-   **첫 초**에 강력한 패턴 인터럽트 존재
-   **안녕하세요/여러분/오늘은** 단어 제거
-   **구체적 숫자** 또는 **시각적 임팩트** 포함
-   **정보 격차**( ) 명확히 제시
-   **초 내** 핵심 가치 제안 완료

**개 모두 체크되어야 등급!**

##  적용 방법 (단계)

### 단계 크루즈 테마 키워드 추출
예 가격, 뷔페, 객실, 액티비티, 노년층 여행

### 단계 가지 핵심 원칙 중 -개 조합 선택

- 내부자 비밀 + 감정 역전
- 충격적 숫자 + 감정 역전

"왜 끝까지 봐야 하는지" 명확히
