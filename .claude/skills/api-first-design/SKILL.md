### 위 -


-
  설명 " →  타입만 생성 (가벼움)"
  주간다운로드 ".+"
  
  적합한경우
    - 타입만 필요하고 클라이언트는 직접 작성할 때
    - 번들 크기 최소화가 중요할 때
    
  사용 
     - ./. - .///.
