##  개발 전 체크리스트


## 개발 전 ( )

-    스펙 정의 완료
-   / 스펙 합의 완료
-   에러 케이스 정의 완료
-   인증/권한 요구사항 정의
-   페이지네이션 전략 결정
-   타입 생성 (--)


##  구현 체크리스트


##  구현

-    적용 (복수형)
-    적용 (인증 필요 시)
-   모든 메서드에 
  -    (자 이내)
  -    (마크다운, 상세)
-   모든 메서드에 
  -   / 성공
  -    검증 오류
  -    인증 오류
  -    권한 오류
  -    리소스 없음
  -    충돌
-    모든  
-    모든  
-      (예시 포함)


##  체크리스트


##  정의

-    모든 필수 필드
-    모든 선택 필드
-    작성
-    제공
-    지정 (, , - 등)
-   검증 데코레이터와 일치 (,  등)
-   - 데코레이터 적용


## 응답 포맷 체크리스트


## 응답 포맷

-   성공   , , ,

-   검증 에러 . 배열 포함
-   페이지네이션  객체 포함


## 배포 전 체크리스트


## 배포 전

-    스펙 유효성 검사 (- )

-    버전 확인
-   이전 버전 호환성 확인 (  시)
-    헤더 설정 (구버전)
