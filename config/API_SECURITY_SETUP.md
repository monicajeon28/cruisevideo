##  체크리스트

### 신규 팀원 온보딩
-   .. 파일 복사
-   필요한  키 발급 받기
-   . 파일 생성 및 키 입력
-   테스트 스크립트 실행 확인
-   . 설정 확인

### 배포 전 확인사항
-   프로덕션 환경 . 파일 생성
-   모든  키 설정 완료
-   파일 권한 설정 ( )
-    키 유효성 테스트
-   로그에  키 노출 없는지 확인

### 정기 점검 (월 회)
-    키 만료일 확인
-   사용량 확인 (무료 한도 초과 여부)
-   의심스러운 활동 모니터링
-   백업 암호화 상태 확인
