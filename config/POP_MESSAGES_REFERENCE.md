## 키워드 트리거 ()

각  메시지는 스크립트의 키워드와 자동 매칭됩니다.

###  예시


/
   "월 만원"
   "월 만원", "월 만원", "하루 만원"
  # 스크립트에 "월 만원"이 있으면 자동 선택

/
   "박일"
   "박일", "박일", "단기"
  # 스크립트에 "단기" 또는 "박일"이 있으면 자동 선택


###  예시


/
   "만원 지원"
   "만원", "지원", "혜택"
  # 스크립트에 "만원 지원"이 있으면 자동 선택

/
   "% 성공"
   "%", "성공", "만족"
  # 스크립트에 "%" 또는 "성공"이 있으면 자동 선택

## 사용 시나리오

### 시나리오   저가 상품 (만원)

**스크립트** "선착순 명, 만원 할인. 월 만원으로 부산출발 개국 여행. 억 보험 기본, 한국어 케어 시간."

**자동 생성 **
. **초** "만원 할인" ( 키워드 매칭)
. **초** "월 만원" ( 키워드 매칭)
. **초** "한국어 케어" ( 키워드 매칭)

---

### 시나리오   프리미엄 (만원)

**스크립트** "선착순 명, 만원 지원. 월 만원에 알래스카 개국, 전부 포함. 년 경력, ,명 선택, % 성공."

**자동 생성 **
. **초** "만원 지원" ( 키워드 매칭)
. **초** "월 만원" ( 키워드 매칭)
. **초** "% 성공" ( 키워드 매칭)

**주의** "월 만원"은 에서 허용 (월 프레임, 가격 분해 아님)

-----------------
